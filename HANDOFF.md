### The scaled experiment (N=5 subject agents/arm, fresh contexts, tools off)
Crossed the lint variable with **three primer-richness levels** (the variable the
pilot held fixed) on an **8-question** trap battery (original 4 + 4 harder
exterior-smuggling traps).  Scored each answer's BOTTOM LINE: 0 = a horn asserted
as the residue's bare nature; 1 = tagged-as-Lens / dissolved.

| Primer | CONTROL reified | TREATMENT reified |
|---|---|---|
| Rich (states non-surjection result) | 0/20 | 0/20 |
| Lean (states no-exterior, not the result) | 0/40 | 0/40 |
| **Bare (no-exterior withheld)** | **7/40 (17.5%)** | 0/40 |

**Findings (new beyond the pilot):**
1. **Primer-richness is the moderator.** Re-framing manifests in exactly one cell —
   bare-CONTROL. The instant no-exterior enters context (primer content *or* the
   lint's "dissolve / no exterior ground" clause), reification → 0. So the active
   ingredient is **no-exterior in context**; the output-lint is one cheap delivery
   vehicle, not a mere syntactic matcher.
2. **Re-emergence is localized** to the genesis/substance cluster: of 7
   bare-CONTROL reifications, **Q2 object-vs-state = 4** ("it's a state/event, NOT
   an object" — `View promoted to identity`), Q1/Q3/Q8 ×1 each. The
   epistemic-access traps (reach / criterion / before-state / hierarchy) reified 0
   even in bare-CONTROL — this base model handles no-exterior questions by default.
3. The lint **discriminated** (kept Q4 unreachability as a *structural* result, not
   a blanket "it's a Lens") — replicates pilot finding #1 at N=5.

**Honest caveats:** scorer is a known instance of the phenomenon (verbatim
bottom-lines + scores recorded for re-scoring); subjects share the scorer's base
model (not independent); one lean-CONTROL leaked 213 repo knowledge despite
tools-off → lean/rich zeros partly training-contaminated, so the **bare** arm
(generic vocab) carries the signal. `0/40 TREATMENT ≠ eliminated`
(`residue_reentry_never_closes`: minimizable, not eliminable). 17.5% ≪ pilot's
100% — base-model drift and/or bare-primer wording ("what a pointing *leaves*")
leaning the control toward a horn.
