# DRLT 26-Parameter Ghost Sum Rule

## 마스터 공식

모든 오차는 하나의 기저 섭동 상수 ε₀에서:

δ(1/α_i) = Sgn_i × (1/α_i)_pred × M_i × ε₀

ε₀ ≈ 0.0038 (현재 우주 연령/호라이즌에 종속)

## 기하학적 가중치 M_i

Strong: M₃ = (n_S²-1)×n_T / n_S = 8×2/3 ≈ 5.33... 
  → 조정: n_S×n_T×(n_S-1)/n_T = ... 
  → 실측 역산: M₃ = 0.47/(8.0×0.0038) = 15.5

Weak: M₂ = n_T×n_S/(n_S+n_T) = 2×3/5 = 1.2
  → 실측: M₂ = 0.40/(30.0×0.0038) = 3.5

EM: M₁ = 1 (기준)
  → 실측: M₁ = 0.22/(59.2×0.0038) = 0.98 ≈ 1 ✓
