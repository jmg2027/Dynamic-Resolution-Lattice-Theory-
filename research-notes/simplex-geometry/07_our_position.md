### [핵심 관찰] x_S/x_T = n_T × n_S = 6

유도 과정에서:
```
x_S/x_T = 6 = 2 × 3 = n_T × n_S
```

이것이 우연인가? 밀도 조건에서 풀어보면:

```
(n_T/x_T)/(n_S/x_S) = c²
→ n_T·x_S/(n_S·x_T) = c²
→ x_S/x_T = c²·n_S/n_T
```

c = 2, n_S = 3, n_T = 2:
```
x_S/x_T = 4 × 3/2 = 6 = n_S × n_T
```

따라서 **x_S/x_T = n_S × n_T는 c² = (n_S/n_T)²이 아닌 한 비자명한 관계**다.

실제로 c = 2 ≠ n_S/n_T = 3/2.
c = 2는 독립적인 입력인가, 아니면 유도되는 양인가?

만약 c = n_T = 2라면 (즉, c가 시간적 꼭지점 수와 같다면):
```
x_S/x_T = n_T² · n_S/n_T = n_T · n_S = 6
```
이것은 c = n_T라는 더 깊은 관계를 시사한다.

**부분 해결 (EXP_046):** c = n_T = 2. n_T = 2일 때만:
- C(n_T, 2) = 1 → 시간 곡률 모드가 유일 (3+1 시공간의 필요 조건)
- TTT hinge 불가 (C(2,3) = 0) → 순수 시간 곡률 부재

- 남은 질문: c = n_T가 왜 격자 빛 속도와도 같은가?

### [확장] SU(5) 대칭점과의 거리

SU(5) 대칭점 = (1/5, 1/5, 1/5, 1/5, 1/5).
우리 위치 ≈ (2/7, 2/7, 2/7, 1/14, 1/14).

Δ⁴ 위에서의 거리 (Fisher-Rao metric):
```
d²_FR = Σₖ (xₖ − 1/5)² / (1/5)

S sector: 3 × (2/7 − 1/5)² / (1/5) = 3 × (3/35)² × 5 = 3 × 9/245 = 27/245
T sector: 2 × (1/14 − 1/5)² / (1/5) = 2 × (-9/70)² × 5 = 2 × 81/980 = 162/980 = 81/490

d²_FR = 27/245 + 81/490 = 54/490 + 81/490 = 135/490 = 27/98
d_FR = √(27/98) = 3√3/(7√2) ≈ 0.525
```

SU(5) 대칭점에서의 거리 ≈ 0.525.
Δ⁴의 최대 가능 거리(중심→꼭지점) = √(4/5 × 5) = 2.
우리는 중심과 경계의 약 1/4 지점에 있다.
