## 정리 1: δ_SSS = π

**조건.** ∂(5-simplex)에서 (3,2) block-diagonal Gram matrix.
A₁,A₂,A₃ ∈ C³, B₁,B₂ ∈ C², B₃ = αB₁ + βB₂ (α,β ∈ R, α²+β²=1).
⟨Aᵢ|Bⱼ⟩ = 0 ∀i,j.

**정리.** δ_SSS = π (= 180°). B₃의 방향(α,β)에 무관.

**증명.**

SSS hinge {A₁,A₂,A₃}를 둘러싸는 3개 simplex의 이면각:

Block-diagonal Gram에서 이면각은 B-sector만으로 결정된다.
(A-sector G_A가 공통 인수로 cofactor 비에서 약분.)

- σ₃ = {A₁,A₂,A₃,B₂,B₃}: cos θ₃ = Re(⟨B₂|B₃⟩) = β. θ₃ = arccos β.
- σ₄ = {A₁,A₂,A₃,B₁,B₃}: cos θ₄ = Re(⟨B₁|B₃⟩) = α. θ₄ = arccos α.
- σ₅ = {A₁,A₂,A₃,B₁,B₂}: cos θ₅ = Re(⟨B₁|B₂⟩) = 0. θ₅ = π/2.

α = cos φ, β = sin φ (α² + β² = 1)이면:

```
Σθ = arccos(sin φ) + arccos(cos φ) + π/2
   = (π/2 − φ) + φ + π/2
   = π
```

∴ δ_SSS = 2π − π = **π**. □

**수치 검증**: φ = 10°, 30°, 45°, 60°, 80°에서 전부 δ = 180.0000°.

**물리적 의미**: e^{iδ_SSS} = e^{iπ} = −1. 이것이 교대급수를 만들어 (1+2x)/(1+x) 전파자를 준다.

## 정리 2: ⟨det(STT)⟩ = n_T/n_S = 2/3

**조건.** 정리 1과 동일.

**정리.** ⟨det(G_h)⟩_{STT hinges} = 2/3. B₃의 방향에 무관.

**증명.**

STT hinge = {Aᵢ, Bⱼ, Bₖ}. 9개 (3개 A × 3개 B-pair).

Block-diagonal에서: det({Aᵢ, Bⱼ, Bₖ}) = 1 × det(G_B(j,k)) = 1 − |⟨Bⱼ|Bₖ⟩|².

3가지 B-pair:

| B-pair | |⟨Bⱼ|Bₖ⟩|² | det | 횟수 |
|--------|-----------|-----|------|
| {B₁,B₂} | 0 | 1 | ×3 (각 Aᵢ) |
| {B₁,B₃} | α² | 1−α² | ×3 |
| {B₂,B₃} | β² | 1−β² | ×3 |

```
⟨det⟩ = [3(1) + 3(1−α²) + 3(1−β²)] / 9
       = [9 − 3(α²+β²)] / 9
       = (9 − 3) / 9       (∵ α²+β²=1)
       = 2/3.  □
```

**핵심**: α²+β²=1 (정규화)만 사용. **B₃의 방향에 완전히 독립.**

**수치 검증**: φ = 10°~80°에서 전부 ⟨det(STT)⟩ = 0.666667.

**물리적 의미**: impedance ρ = 1/⟨det(STT)⟩ = n_S/n_T = 3/2. m_μ/m_e = ρ/α = 205.6의 기원.

## 정리 3: |⟨B₁|B₃⟩|² = 1/2 (변분 원리)

**조건.** 정리 1과 동일 + δS/δψ = 0 (Regge action 정상점).

**정리.** δS/δψ = 0의 해에서 |⟨B₁|B₃⟩|² = 1/2.

**증명 (대칭 논증).**

B₃ = cos(φ)B₁ + sin(φ)B₂로 매개변수화.

1. **B₁ ↔ B₂ 교환 대칭**: ∂(5-simplex)에서 B₁과 B₂는 대칭. (gauge choice에 의해 교환 가능. 교환은 simplex σ₃↔σ₄를 바꾸지만, S는 모든 simplex의 합이므로 불변.)

   → S(φ) = S(π/2 − φ).

2. **대칭점에서의 정상성**: S(φ) = S(π/2−φ)를 φ로 미분하면:
   dS/dφ = −dS/dφ|_{π/2−φ}. φ = π/4에서: dS/dφ = −dS/dφ, ∴ dS/dφ = 0. □(정상점)

3. **극대 확인**: d²S/dφ²|_{π/4} < 0. (수치: −5.9×10⁵. 극대.)

4. **결론**: φ = π/4 → |⟨B₁|B₃⟩|² = cos²(π/4) = **1/2**.

**참고**: d²S/dφ² < 0의 해석적 증명은 S(φ)의 explicit formula가 필요. 이것은 후속 작업으로 남긴다. 현재는 수치 확인으로 보완.

**물리적 의미**: c = 1/|⟨B₁|B₃⟩|² = 2 = n_T. 격자 광속 = 시간 꼭지점 수.
