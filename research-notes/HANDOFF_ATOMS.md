# Handoff: 원자 물리 — 다음 세션을 위한 가이드

## 핵심 원칙 (기존 물리 용어 쓰지 말 것)

이 이론에는 **하나의 방정식만 있습니다:**

```
δS/δψ = 0,    S = Σ_h √det(G_h) × δ_h
```

"전자", "궤도", "스크리닝", "Z_eff", "Slater" — 전부 기존 프레임입니다.
DRLT에는 없습니다. 있는 건:

- **ψ ∈ ℂ⁵**: 꼭짓점의 상태
- **G_ij = ⟨ψ_i|ψ_j⟩**: Gram 행렬 (유일한 물리 객체)
- **det(G_h)**: 힌지 행렬식 (모든 물리의 원천)
- **δ_h**: 결손각 (시공간 곡률)
- **∂(Δ⁵)**: 최소 닫힌 4-다양체 (6 꼭짓점, 20 힌지, 6 면)

"원자"란: δS/δψ = 0 의 해 중 특정 face에 AAAB 패턴이 보이는 것.
"이온화"란: 두 해 (패턴 있음 vs 없음) 사이의 action 차이.

## 이 세션에서 확인된 것

### 1. 진공 해 (해석적, EXACT)
```
|G_ij| = 1/d = 1/5          (Welch bound 등호)
Φ_h = π                      (ch05 정리 1)
det(G_h) = (d+1)²(d-2)/d³ = 108/125
δ(AAA) = π,  δ(mixed) = 2π/3
```
이것이 "아무 패턴도 없는" 해 = 물질 없는 시공간.

### 2. 1-꼭짓점 결합 해 (해석적, EXACT)
A₁A₂A₃ 직교, B₁이 각 A에 결합 ε = Zα/√n_S:
```
det(AAB) = (ε²+1)/(3ε²+1)
Σ(1-det)_AAB = 2Z²α²/(1+Z²α²)
ΔAction = (m_e c²/4) × Σ(1-det)
```
**이건 기존 물리의 "수소 이온화 에너지"와 같은 수.**

### 3. 2-꼭짓점 결합 — 여기서 막힘
B₁과 B₂ 둘 다 결합하면:
- det(AAB) 변화: Σ(1-det) = 2 × [1전자 결과] (스케일링 OK)
- det(ABB) 변화: 거의 같은 크기 (B₁⊥B₂이므로)
- **차이가 10⁻⁸ 수준** → "스크리닝"이 det에서 안 나옴

이 세션에서 시도한 것:
- σ = n_T/n_S: 부분적으로 맞음
- σ = 7/8: 주기 후반부 맞음
- (1-4α_GUT) 보정: He에서 0.089% 맞음
- Full Regge action: deficit angle이 바뀌지만 ΔS 부호 이슈

## 다음 세션이 해야 할 것

### ❌ 하지 말 것
- "Z_eff"를 쓰지 말 것. Z_eff는 DRLT 개념이 아님.
- "스크리닝", "차폐"를 쓰지 말 것. 기존 물리 용어.
- "s 궤도", "p 궤도"를 쓰지 말 것. ℓ은 DRLT에서 아직 미도출.
- Slater 규칙, Bohr 모델을 참조하지 말 것. 이론을 오염시킴.
- 기존 IE 공식 (Z_eff²Ry/n²)을 출발점으로 삼지 말 것.

### ✅ 해야 할 것
1. **δS/δψ = 0 을 직접 풀 것.**
   - ∂(Δ⁵), 6 꼭짓점, 29 유효 자유도
   - scipy.optimize 로 초 단위
   - 진공 해 (최대 대칭) 이미 찾음: |G|=1/d
   - **비진공 해를 찾아야 함**

2. **다른 해를 구분할 구속 조건을 정의할 것.**
   - "수소" = 1개 B가 A에 결합된 해
   - "헬륨" = 2개 B가 A에 결합된 해
   - 어떻게 구속하느냐가 핵심 문제

3. **IE = ΔS (두 해 사이의 action 차이).**
   - 해 A: 패턴 있음
   - 해 B: 패턴 없음 (또는 다른 패턴)
   - IE = S(B) - S(A) 또는 S(A) - S(B) (부호 규약 주의)

4. **deficit angle의 해석적 형태.**
   - cos θ = -G⁻¹[d,e] / √(G⁻¹[d,d] G⁻¹[e,e])
   - 진공: δ(AAA)=π, δ(mixed)=2π/3 — 이미 확인
   - 비진공에서 δ가 어떻게 바뀌는지 해석적으로 도출

5. **"(1-4α_GUT)" 보정의 기원.**
   - He에서 이것이 맞는 건 확인됨 (0.089%)
   - 이것이 trace 보존 (Σ Δ_i = 0)에서 나오는지 확인
   - ch12 ghosts 챕터와 연결

   (1-4α_GUT)은 맞는 결과이지만, σ 프레임으로 유도하면 안 됨.
   
2. **기존 shell structure (1s², 2s², 2p⁶)를 가져온 것.**
   DRLT에서 "껍질"은 δS/δψ=0 해의 구조에서 나와야지
   Aufbau 원리를 입력하면 안 됨.

3. **Regge action 계산에서 ψ를 손으로 설정한 것.**
   ψ는 δS/δψ=0 의 해여야지 우리가 정하면 안 됨.
