## Bool = minimal self-reference

In Note 24 §7, "Bool has no bootstrap (depth 1)" was stated.
This is correct only in the narrow sense that **there is no Nat-style
bootstrap**, but **Bool already contains the more fundamental self-reference**.

Bool's self-reference:
- Two elements {true, false}, each the negation of the other.
- `¬` operation has no fixed point (x = ¬x impossible).
- Bivalent oscillation of the liar "this statement is false".

Nat's self-reference:
- Nat requires Nat in its definition (bootstrap).
- Fixed point of the Lambek initial algebra.

**Both are manifestations of Raw's self-reference through different Lenses.**
The essence is the same — the axiom itself is self-reference.

**Raw.a, Raw.b are themselves structurally isomorphic to Bool**: 2-element,
mutually distinguishing.  Primitive distinction = primitive negation.

Therefore "Bool has no bootstrap" is a false dichotomy.  **Every Lens
contains self-reference** — because Raw does.
