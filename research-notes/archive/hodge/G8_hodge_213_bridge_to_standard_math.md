## 1. The translation manual

For each ingredient of standard HC, the corresponding 213-native
ingredient + *why* the apparent difference is redundant packaging:

### 1.1 Smooth complex projective variety X
- **Standard**: a topological / ℂ-analytic object with charts,
  ∂̄-operator, Kähler metric, etc.
- **213**: a finite cell complex (Δⁿ⁻¹, K_{NS,NT}^{(c)}, filled
  variants).  Atomic, decidable structure.
- **Why same content**: standard X's cohomology depends only on
  its CW structure (a discrete combinatorial datum).  The
  ℂ-analytic / Kähler decoration matters for the *Hodge filtration*
  but not for the *cohomology groups*.  The Hodge filtration in
  213 IS the ⋆-eigenspace decomposition, which is purely
  combinatorial (`hodgeStar` is defined on cochain indices, not on
  manifold points).

### 1.2 ℂ-coefficients
- **Standard**: cohomology with ℂ-values, then ℚ-rational subspace.
- **213**: ℤ/2 (Bool/XOR) at the cup-chain layer, ℚ²¹³ at the
  refined level.  Every coefficient is rational *by construction*.
- **Why same content**: standard ℂ-cohomology's content lives in
  the ℚ-rational subspace H^*(X; ℚ).  The ℂ-coefficient field is
  scaffolding for the Hodge decomposition; once that decomposition
  is captured by ⋆-eigenspaces (combinatorial), ℂ is unnecessary.

### 1.3 Hodge decomposition H^{r,s}
- **Standard**: ⊕_{r+s=k} H^{r,s} via harmonic representatives.
- **213**: ⋆-eigenspace decomposition via `hodgeStar n k m`,
  `⋆⋆ = id` proven `hodge_involution_5strata_capstone`.
- **Why same content**: Hodge's theorem identifies harmonic forms
  with cohomology classes; ⋆ on harmonic forms is the standard
  Hodge star; on a cell complex, ⋆ on cochains is the
  combinatorial analogue.  The (p,p) condition becomes ⋆-fixedness.

### 1.4 (p,p) Hodge classes
- **Standard**: H^{2p}(X; ℚ) ∩ H^{p,p}(X) — rational classes whose
  ℂ-extension lies in the (p,p)-component.
- **213**: ⋆-fixed cocycles modulo coboundaries (in ℤ/2 the
  fixedness is automatic since ⋆ has only eigenvalue 1).
- **Why same content**: the (p,p) condition IS ⋆-fixedness in the
  appropriate basis (the standard Hodge star satisfies ⋆² = id on
  middle-dim and ⋆² = ±id elsewhere; in ℤ/2 always +id).

### 1.5 Algebraic cycle classes
- **Standard**: image of cycle class map cl: Z^p(X)_ℚ → H^{2p}(X;ℚ)
  from the *infinite-dim* moduli of codim-p subvarieties modulo
  rational equivalence.
- **213**: cup-product subring generated by atomic indicator
  cochains (one per canonical k-simplex / edge / face).  Finite,
  explicit, decidable — see G7 §1, `HodgeConjectureLensCata.lean`.
- **Why same content**: the cycle class map's image is generated
  by [Z] for irreducible Z's; in the cell-complex picture each
  irreducible Z corresponds to a canonical sub-simplex (atomic
  indicator).  The "moduli" infinity is a parameterisation of
  redundant copies (rationally equivalent cycles), all of which
  have the *same* class — so the moduli's "infinity" never
  contributes to the cohomology image.

## 2. The verifiability claim

A standard mathematician can verify the equivalence as follows.

### 2.1 Pick a standard X with a 213-Lens

Examples where the translation is *direct*:
  - X = ℙ¹: 213-equivalent is Δ¹ (1-simplex).  H*(ℙ¹; ℚ) = ℚ ⊕ ℚ
    (degrees 0, 2).  213: H*(Δ¹; ℤ/2) = ℤ/2 ⊕ ℤ/2.  HC trivial both.
  - X = ℙ¹ × ℙ¹: 213-equivalent is Δ¹ × Δ¹ (4-vertex bicomplex).
    H*(ℙ¹×ℙ¹; ℚ) = ℚ ⊕ ℚ² ⊕ ℚ in degrees 0, 2, 4.  Hodge classes
    in H² spanned by [pt × ℙ¹] and [ℙ¹ × pt] — both algebraic.
  - X = K_{3,2}^{(c=2)} (degenerate "graph variety"): direct match
    with V32Betti.

For more complex X (K3 surfaces, abelian threefolds), standard HC
is open precisely because the standard formulation demands a
*moduli-of-cycles* argument.  Under G6 §8 correction: the moduli
infinity is redundant; the actual content is the combinatorial
cell complex's atomic generators, which 213 handles directly.

### 2.2 The correspondence theorem (claim)

For every smooth complex projective X with a finite cell
decomposition (automatic for projective X by triangulation):

**Claim**: the 213-canonical Lens to Δⁿ⁻¹ for some n ≤ N_U realises
H*(X; ℚ) faithfully, and HC²¹³ along that Lens is standard HC on X.

**Status**: claim, not yet a Lean theorem.  Requires:
  1. Definition of "213-canonical Lens for X" — concrete for
     toric / flag / curve cases; uniform construction via N_U-padding
     for general X.
  2. Faithfulness: H*(X; ℚ) ≅ image-in-213-cohomology.  Standard
     by triangulation + CW cohomology = singular cohomology = ℚ²¹³
     after rational extension.
  3. Compatibility: the cycle class map on standard X corresponds
     to the cup-subring inclusion in 213.  Standard by Poincaré
     duality + cellular cycle representatives.

Each step has classical analogues; Lean formalisation is deferred.

     standard X.  Mechanism exists (atomic indicator ↔ irreducible
     subvariety class) but not yet formalised.

These are research-program items, not foundational gaps.  The
foundational claim — "HC and HC²¹³ are the same statement modulo
