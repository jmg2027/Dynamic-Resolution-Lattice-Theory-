## Priority A: 공식 자체가 잘못된 것 (구조적 수정 필요)

### A1. Higgs 질량 m_H — 121 vs 125 GeV

**현재 공식:** `m_H = v_H √(c·α_GUT·d)`

**문제:** v_H = 246 GeV일 때 √(2×0.0243×5) = 0.493이므로 m_H ≈ 121 GeV.
관측값 125.25 GeV와 3% 차이. Ξ 보정(0.02%)으로 해결 불가.

**가설:**
- √(c·α·d) 대신 √(c·α·(d+1)) → √(2×0.0243×6) = 0.540 → 133 GeV (과대)
- 계수가 c·α·d가 아닌 다른 조합론적 인자일 수 있음
- self-coupling λ의 유도가 불완전할 가능성

**EXP:** EXP_059_ckm_precision.py에서 확인됨
**파일:** ch07_mixing.tex, Eq. m_H

### A2. Δm_np 기하 인자 — +11%

**현재 공식:** `Δm = (m_d-m_u) × n_S(d-1+S₂/S∞)/d²`

기하 인자 = 0.571, 관측에서 역산하면 0.513 필요. 비율 = 0.897.

**문제:** 기하 인자가 ~10% 과대. Ξ 보정(0.85%)으로 해결 불가.

**가설:**
- S₂/S∞ = 15/(2π²) 계산에 빠진 보정이 있을 수 있음
- `n_S(d-1)/d²` = 0.48이면 오히려 -6% 과소
- 중간값 0.513에 해당하는 조합론적 표현이 있을 수 있음
  - n_S(d-1)/d² × (1 + α/(n_S-1)) ≈ 0.486 → 여전히 부족
  - 새로운 기하 인자 필요

**EXP:** EXP_061_np_mass_diff.py
**파일:** ch06_masses.tex, Eq. Δm_np

### A3. 1/α₂ (약한 결합 상수) — 18.2 vs 29.6

**현재 공식:** `1/α₂ = 12·n_T·S₂/S∞`

이것은 GUT scale 값이 아니라 저에너지 running이 포함되어야 함.
현재 공식에는 running이 없음.

**가설:** 완전한 β-function running 포함 필요.

**EXP:** EXP_062_remaining.py
**파일:** ch05_couplings.tex

### A4. η_B (바리온 비대칭) — 공식 불일치

CLAUDE.md에는 "6.10×10⁻¹⁰, 0.04% 오차"라고 기록되어 있지만,
`6α³/(π²φ) = 5.4×10⁻⁶`으로 관측값의 10⁴배.
ch09_cosmology.tex의 실제 공식은 다를 수 있음.

**필요:** ch09_cosmology.tex에서 정확한 η_B 공식 확인 및 검증.

**EXP:** EXP_062_remaining.py
**파일:** ch09_cosmology.tex

### A5. Ω_Λ (암흑 에너지) — 공식 불일치

## Priority B: 정량적으로 부족한 것 (미세 조정 필요)

### B1. 중성미자 질량비 — 3.73 vs 5.71

DRLT seesaw (D×T⁻¹×D) 구조가 283→3.73으로 개선했지만 관측 5.71과 35% 차이.

**가설:**
- T 행렬의 고차 보정 (α_GUT 차수)
- det_B 가중치의 미세 조정
- M_R 구조의 비대각 원소
- 가능한 접근: T + α_GUT·T₂ (perturbative T correction)

**EXP:** EXP_054_neutrino_ratio.py
**파일:** ch07_mixing.tex §Neutrino masses

---

### B2. sin²θ₂₃ — 보정 방향

1항(+2α) 후 +0.13σ. 추가 보정은 μ-τ 대칭 보호 때문에 방향 결정이 미묘.
현재 EM 복원 항(-α_em·α·n_T)으로 +0.11σ까지 개선했지만 물리적 유도 불완전.

**필요:** B-sector 대칭 깨짐의 정확한 차수 결정.

### B3. 1세대 쿼크 질량 m_u, m_d — Ξ 보정 악화

Ξ 보정이 m_u를 -1.06% → -1.08%로 약간 악화.
1세대는 confined coupling (x = -ε)이 지배적이라 범용 Ξ가 부적합.

**가설:** 1세대 전용 Ξ_confined 필요.
