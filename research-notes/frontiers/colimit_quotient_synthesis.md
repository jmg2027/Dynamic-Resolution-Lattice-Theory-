## The verdict — one name, two objects

A quotient by an equivalence *generated by local moves* (Reidemeister/Markov for knots, homotopy moves for
π₁) is a **rewriting system**, and "quotient set = set of normal forms" holds **exactly when the system is
confluent + terminating** (Newman + Church–Rosser). That test cleaves the corner:

### Side A — INTERFACE DEFECT (buildable ∅-axiom *now*, no `Quot.sound`)
When the move-system is **confluent + terminating**, the quotient is the **normal-form subtype**
`{t // Irreducible t}` + a `normalize` section — exactly how the repo already represents canonical objects,
**no `Quot`**. The decisive grounding (verified PURE this session): the repo *already ships* `Quot.sound`'s
content axiom-free — `Lens.Unified.LensImage` (the quotient-as-Σ-type) with
**`LensImage.proj_val_eq_iff : (proj x).val = (proj y).val ↔ L.equiv x y`** (`Unified.lean:163`,
`#print axioms` → **no axioms**; the file documents "no `Quot.sound` required"). Termination is in hand:
`Lambek.no_infinite_descent`/`isPart_wf`/`part_depth_succ_le` (explicit ℕ-measure, the strong-normalization
floor of `curry_howard.md`); confluence-for-free via `dhom_unique_pointwise` initiality; the reduction order
+ decidable "same normal form" via `Lens.refines`/`lensIso_iff_kernel_eq`. **Buildable cases:** the braid
group `Bₙ` (Garside greedy normal form), free / surface-group / complete-rewriting π₁ (free reduction), knot
classes with a canonical diagram, and the **Kauffman-bracket Jones state sum** (a terminating
crossing-resolution fold). Constructive-topology framing: the same is the *witnessed-deformation groupoid*
(carry the homotopy/isotopy **as data** — a path-with-modulus — exactly as `Real213` carries a real as a
regular sequence, not a Cauchy quotient; `holonomy_append`/`holonomy_nil` = the groupoid composition/identity,
`lensIso_refl/symm/trans` = the equivalence-of-witnesses layer). This is the **constructive wall in
topological clothing**, dissolved the same way: a bare `∃`-relation quotiented by `Quot.sound` ↦ a
witness-carrying equivalence, ∅-axiom.

### Side B — GENUINE OBSTRUCTION (theorem-grade, irreducibly `Quot`)
When the move-system is **non-confluent / the word problem undecidable**, *no* normal-form type can exist —
a `normalize` would decide the undecidable. This is not missing optimism; it is **theorem-grade**:
- **General / arbitrary-presented π₁** — the **Novikov–Boone theorem**: the word problem for finitely
  presented groups is *undecidable*. A `normalize : Word → NormalForm` would solve it, so it cannot exist
  ∅-axiom (or otherwise).
- **General Reidemeister knot equivalence** — decidable, but with *no* monotone confluent system (unknotting
  must sometimes *raise* crossing number, so no `crossing-count` measure terminates).
- **The ambient-S³ isotopy quotient** — there is no `Raw`/`Lens` term type for the ambient 3-manifold at
  all (missing *modelling input*, orthogonal to the quotient machinery).

`Quot.sound` is axiom-dirty, so Side B is a real ∅-axiom boundary — and the sharpest possible witness that
the corner is *genuinely* open is a classical undecidability theorem sitting under it.
