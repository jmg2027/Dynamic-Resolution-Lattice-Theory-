# The colimit / ambient-deformation quotient, from the rewriting / normal-form angle

*Frontier memo. One of a 3-school panel attacking the single recurring OPEN corner of the 213
Decomposition Calculus: the **ambient-deformation / colimit quotient** (isotopy classes of knots,
homotopy classes of loops in `π₁`). Located in `SYNTHESIS.md` §5.1 + `practice/knots.md` Gap 2 +
`practice/fundamental_group.md` leg 4 + `practice/two_cells.md` Shape 3. This school's home turf:
**a quotient by a generated relation IS a rewriting system, and "the quotient set" = "the set of
normal forms" exactly when the rewriting is confluent + terminating (Church–Rosser + Newman).** All
Lean names below grep-verified in `lean/E213/`.*

---

## 1. The diagnosis: every one of these quotients is a quotient-by-a-rewriting-system

The recurring corner is always the same categorical object — a coequalizer / colimit, i.e. a set
modded by the relation **generated** by a family of local moves. Term-rewriting is the theory of
exactly that object. Map each field's quotient to its rewriting system (the moves) and its
to-be-computed normal forms (the quotient representatives):

| Field | Underlying terms | Local moves (the rewrite rules) | Quotient = ? |
|---|---|---|---|
| **knot isotopy** | knot diagrams | the three **Reidemeister moves** | isotopy classes |
| **braid equivalence** | braid words in `Bₙ` | braid relations `σᵢσⱼ=σⱼσᵢ` (\|i−j\|≥2), `σᵢσᵢ₊₁σᵢ=σᵢ₊₁σᵢσᵢ₊₁` | braid group elements |
| **link equivalence** | braid closures | braid relations **+ Markov moves** (conjugation, (de)stabilization) | link types |
| **π₁** | loop words (edge-paths / `holonomy` products) | **homotopies as 2-cell rewrites** (path × `[0,1]`) | homotopy classes `[γ]` |
| **knot *invariant* (Jones)** | unoriented diagrams | **Kauffman-bracket skein resolution** `L → L₀ + L∞` | bracket polynomial (a state sum) |

The unifying statement: **a residue the calculus "cannot generate from any reading's kernel"**
(`knots.md` Gap 2, `fundamental_group.md` leg 4) is precisely a quotient whose relation is
*generated by local moves* rather than read off as a kernel `lensIso_iff_kernel_eq`. A kernel
quotient is decided in one shot (`decide (s=r)`, `FlatOntology.Object1`); a move-generated quotient
is the *reflexive-transitive-symmetric closure* of a one-step relation — which is exactly the object
a rewriting system computes. So the corner is not mysterious: it is the **word problem** for the
rewriting system of moves.

### The ∅-axiom reformulation

The CLAUDE.md hard rule is: **no `Quot`.** A naive quotient `Term / (generated relation)` needs
`Quot.sound` (axiom-dirty). The rewriting-theory reformulation dodges `Quot` *when and only when* the
rewrite system is **confluent + terminating**:

> If a rewrite relation `↝` on a `Term` type is terminating (well-founded, no infinite `↝`-chain)
> and confluent (any two reduction paths from `t` reconverge), then by **Newman's lemma** every term
> has a **unique normal form** `nf(t)`, and the function `normalize : Term → Term` lands in the
> **subtype** `{ t // Irreducible t }`. The quotient set is then *represented* by this subtype with
> `normalize` as the section — **no `Quot`, no `Quot.sound`.** Two terms are equivalent iff
> `normalize t = normalize u`, a *decidable* `Bool`/`decide` equality, the q=+1 PURE corner
> (`SYNTHESIS.md` §6, the PURE/DIRTY = Heyting/Boolean boundary).

This is **exactly how the repo already represents canonical objects**: a canonical reduced form +ed
a fold-to-it. The whole calculus's normal-form thesis `OBJECT = ⟨C|L⟩` (`SYNTHESIS.md` §1) is a
normal-form representation; `curry_howard.md` makes it explicit — `Raw.fold` *is* the recursor,
normalization *is* the fold to the unique normal form (`dhom_unique_pointwise`), and strong
normalization is **already built**: `Lambek.no_infinite_descent` / `Lambek.isPart_wf`.

So the panel's question reduces to a single sharp test, **per field**: *is the move-system confluent
and terminating?*
- **Yes** ⟹ INTERFACE DEFECT (build the normal-form subtype + `normalize`, ∅-axiom, no `Quot`).
- **No** ⟹ GENUINE OBSTRUCTION (no normal-form type exists; the quotient is irreducibly a `Quot` of
  an undecidable / non-confluent relation).

## 2. The repo already furnishes the confluent side's engine

The decisive observation: the calculus's own termination theorem is a **termination order in the
exact sense rewriting theory needs**, and it is ∅-axiom PURE.

`Theory/Raw/Lambek.lean` (grep + Read-verified, 30/0 PURE per `curry_howard.md` scan):
- `isPart_wf : WellFounded IsPart` (`:199`) — the peel/reduction relation is well-founded.
- `part_depth_succ_le (c p) (h : IsPart c p) : c.depth + 1 ≤ p.depth` (`:245`) — **each rewrite step
  strictly drops the `depth` measure by at least the unit `1`.** This is a *reduction order*:
  a monotone `ℕ`-valued measure strictly decreasing under `↝`.
- `descent_chain_drops (chain) (hstep) : ∀ k, (chain k).depth + k ≤ (chain 0).depth` (`:252`) — the
  explicit rate: after `k` steps depth has dropped by `≥ k`.
- `no_infinite_descent (chain) (hstep) : False` (`:273`) — **no infinite reduction sequence**: a
  chain of length `(chain 0).depth + 1` would force `depth+1 ≤ depth`. This is **strong
  normalization (SN)**, term for term, by an explicit length bound, not abstract accessibility.

This is the **terminating half of Church–Rosser delivered, ∅-axiom**, for the calculus's own term
algebra. The pattern generalizes: *any* rewrite system equipped with a strictly-decreasing
`ℕ`-valued measure (a "depth" / weight / length) reduces termination to `no_infinite_descent`'s
shape — strong induction on the measure, no `Classical`, no `Quot`. The repo also supplies the
*confluence-side* primitives:
- `Raw.fold` (`Theory/Raw/Fold.lean:22`) + `fold_a/fold_b/fold_slash` (`:30,:33,:37`) — the
  catamorphism: a structurally-recursive `normalize`-shaped evaluator, definitionally computing on
  every constructor (`rfl`-checked). This is the `normalize` function's archetype.
- `dhom_unique_pointwise` (`Lens/Foundations/UniversalDistinguishing.lean:103`) — **initiality
  forces uniqueness**: any two structure-maps out of the free object agree pointwise. This is the
  *confluence-for-free* fact for the calculus's own terms: the normal form is unique because the
  arrow out of the initial object is unique.
- `Lens.refines` / `refines_trans` (`Lens/LensCore.lean:90,95`) — the reduction *order* itself
  (a thin category: 1-cells = refinement steps, composition = transitivity), and
  `lensIso_iff_kernel_eq` (`Lens/Unified.lean:64`) — the *decidable* "same normal form" relation
  (kernel coincidence) the confluent quotient collapses to.

**Conclusion of §2:** for any rewrite system that is confluent + terminating, the repo has every
ingredient to build the quotient as a normal-form subtype `{ t // Irreducible t }` with a `normalize`
section, with termination delegated to a `no_infinite_descent`-style measure argument and uniqueness
to a `dhom_unique_pointwise`-style initiality/confluence argument — **all ∅-axiom, no `Quot`.**

## 3. The SPLIT (the honest verdict — it does not collapse to one answer)

The verdict splits cleanly along the confluent+terminating line. This is the rewriting school's
sharpest contribution: the recurring corner is **not one object** — it is two, and they fall on
opposite sides.

### Side A — INTERFACE DEFECT (confluent + terminating ⟹ normal-form subtype, ∅-axiom, no `Quot`)

These quotients **have a solvable word problem via a canonical-form algorithm**, so the quotient set
*is* a normal-form type and is buildable with the §2 engine:

1. **The braid group `Bₙ` word problem.** Braids have a *solvable* word problem; the braid monoid
   has a confluent terminating rewriting presentation (Garside normal form / greedy form — every
   braid has a unique left-greedy decomposition). So `Bₙ`-equivalence is `normalize`-decidable: the
   quotient `braid words / braid relations` IS the subtype of Garside-normal-form words +
   `normalize`. **No `Quot`.** The repo already reaches the permutation image `Bₙ → Sym(n)`
   (`knots.md`: `PermGroup.composeList`, `Zolotarev.mulPerm_comp` Cayley) and the braid relator at
   the Coxeter level (`(ts)³=e`, `C3ChainCapstone.c3_chain_master` (f)); the missing piece is the
   *pure-braid kernel*, but that kernel too is Garside-confluent — so it is buildable, not blocked.
   Termination of the Garside rewriting reduces to a length measure → `no_infinite_descent` shape.
2. **Knot/link classes with a canonical diagrammatic normal form.** Sub-families where a confluent
   terminating diagrammatic reduction exists — e.g. classes detected by a *complete* computable
   invariant, or alternating knots via the (proven) flype/Tait machinery — give canonical
   representatives, hence a normal-form subtype.
3. **The Kauffman-bracket / Jones *state sum* itself.** `two_cells.md` Shape 2 already grounds the
   skein's *algebraic form* as the **graded-relation slot** (`leibniz_universal_delta4`, PURE). The
   bracket is computed by a **terminating** crossing-resolution recursion: each resolution `L → L₀,
   L∞` strictly drops the crossing count — a `depth`-style measure, exactly the
   `descent_chain_drops` shape. The bracket polynomial is therefore a `normalize`-style fold over a
   well-founded resolution tree → ∅-axiom *as a polynomial-valued function*, no `Quot`. (Its
   *invariance* under Reidemeister moves is the separate, harder claim — side B.)
4. **π₁ of spaces with a confluent presentation.** Where `π₁` is given by a group presentation with
   a complete (confluent + terminating) rewriting system — free groups (free reduction: cancel
   `aa⁻¹`, a terminating length-dropping confluent system), surface groups, more generally groups
   admitting a finite complete rewriting system — the homotopy quotient `loops / homotopy` IS the
   normal-form subtype of reduced words. The repo's `holonomy_append` (loop concatenation = monoid
   hom, `HolonomyLattice.lean:108`) furnishes the term monoid; free reduction furnishes the
   confluent terminating system; the quotient is a subtype, **no `Quot`.**

For every Side-A case the build sketch is the same:
```
  Term            : the word / diagram type (e.g. List of generators)
  step  ↝         : one local move that strictly drops a ℕ-measure μ (length / crossing / depth)
  termination     : WellFounded ↝   — strong induction on μ, the no_infinite_descent shape
  confluence      : local confluence + Newman (μ-well-founded) — critical-pair check, decidable
  Irreducible t   : ¬ ∃ u, t ↝ u    — decidable
  normalize       : Term → {t // Irreducible t}   — iterate ↝ (terminates by μ)
  quotient ≃      : t ~ u  :=  normalize t = normalize u   — DECIDABLE Bool-equality (q=+1, PURE)
```
This never touches `Quot`/`Quot.sound`/`Classical`. It is the q=+1 PURE corner of `SYNTHESIS.md` §6.

### Side B — GENUINE OBSTRUCTION (non-confluent / undecidable ⟹ no normal-form type; irreducibly `Quot`)

These quotients have **no confluent terminating rewriting system** — either provably (undecidable
word problem) or with no known complete system — so **no normal-form subtype exists**, and the
quotient is irreducibly a `Quot` of a relation that is not `decide`-able:

1. **General homotopy / π₁ of an arbitrary presented space.** The word problem for general finitely
   presented groups is **undecidable** (Novikov–Boone). When `π₁` is such a group, *no* `normalize`
   function can exist (it would decide an undecidable problem), so the homotopy quotient *cannot* be
   a normal-form subtype with a computable section. It is irreducibly `Quot (loop words) (homotopy)`
   over an undecidable relation. This is a **theorem-grade obstruction**, not a missing engine.
2. **General Reidemeister equivalence (the full unknotting / knot-equivalence problem).** Knot
   equivalence is *decidable* (Haken/Hemion, Kuperberg), but there is **no simple confluent
   terminating Reidemeister rewriting system**: Reidemeister moves are *not* monotone (the unknotting
   sequence may have to *increase* crossing number before decreasing — the Goeritz / hard-unknot
   phenomenon), so no `crossing-count`-style measure makes the system terminating, and the system is
   wildly non-confluent (no canonical diagram). Decidability comes from a *different* (normal-surface)
   machinery, not from confluent rewriting. So the isotopy quotient on diagrams is **not** a
   normal-form type via the §2 engine; representing it ∅-axiom would require importing the entire
   normal-surface decision procedure as a `normalize` — far beyond a normal-form subtype, and the
   *quotient as such* still wants `Quot`.
3. **The full ambient-isotopy quotient in `S³` (the "ambient-space construction").** Even granting a
   decision procedure, the object `knots.md`/`fundamental_group.md` name — *continuous deformation in
   an ambient 3-manifold* — is not a rewriting system on a `Raw`/`Lens` term at all. There is no
   `Term` whose local moves generate it; the ambient space is the missing primitive
   (`two_cells.md` Shape 3: "absent ambient-space construction" + "un-built colimit/q=−1 corner").
   This is the deepest genuine absence: not even a non-confluent rewriting *presentation* is in hand,
   because the term type itself is absent.

The unifying obstruction statement: **a quotient is representable ∅-axiom as a normal-form type iff
its generating move-system is confluent + terminating; for the undecidable / non-confluent cases the
quotient is provably *not* a normal-form type (a `normalize` would decide the undecidable), so it is
irreducibly `Quot` — and `Quot.sound` is axiom-dirty (CLAUDE.md hard rule). Side B is therefore a
genuine boundary of the ∅-axiom calculus, not an interface defect.**

## 5. Headline verdict (6–10 sentences)

The single recurring open corner — the ambient-deformation / colimit quotient (knot isotopy,
braid/link equivalence, π₁ homotopy classes) — is, from the rewriting-theory angle, exactly a
**quotient by a relation generated by local moves**, i.e. a rewriting system, and "the quotient set"
equals "the set of normal forms" precisely when that system is **confluent + terminating** (Newman +
Church–Rosser). The verdict therefore **SPLITS, and does not collapse to one answer.** On the
**confluent+terminating side — INTERFACE DEFECT, ∅-axiom buildable with no `Quot`**: the braid group
word problem (Garside greedy normal form), free / surface / complete-rewriting-system `π₁`'s (free
reduction), knot classes with a canonical diagram, and the Kauffman-bracket *state sum* itself (a
terminating crossing-resolution fold) — each is representable as a normal-form **subtype**
`{t // Irreducible t}` + a `normalize` section, with termination delegated to a
`no_infinite_descent`-style ℕ-measure argument (already PURE-built in `Lambek.lean`) and uniqueness
to `dhom_unique_pointwise`-style initiality, exactly how the repo already represents canonical
objects (q=+1 PURE corner). On the **non-confluent / undecidable side — GENUINE OBSTRUCTION**:
general homotopy / arbitrary presented `π₁` (Novikov–Boone *undecidable* word problem — a
`normalize` cannot exist), general Reidemeister equivalence (decidable but with *no* monotone
confluent system — unknotting must sometimes raise crossing number), and the full ambient-isotopy
quotient in `S³` (no `Raw`/`Lens` term type for the ambient space at all) are provably **not**
normal-form types; their quotient is irreducibly `Quot` of a relation that is not `decide`-able, and
`Quot.sound` is axiom-dirty — so Side B is a real ∅-axiom boundary, not an interface gap. The
rewriting school's clean contribution is that the "single recurring absence" was always **two
objects wearing one name**: a confluent-rewriting quotient that the calculus's own termination
engine already reaches, and an undecidable / non-confluent quotient that no normal-form type can
capture. Do not force optimism: braids and the bracket fall, but knot equivalence and general π₁ do
not — and the latter's resistance is theorem-grade (undecidability), the strongest possible witness
that the corner is genuinely, not interfacially, open.
