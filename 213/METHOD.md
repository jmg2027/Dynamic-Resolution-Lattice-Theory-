# 213 방법론

## 절차

어떤 문제 X에 대해:

### Step 0. 213 비용 판정
X의 비용을 측정한다. 비용 ≤ 5이면 진행. > 5이면 구조화 필요.

### Step 1. 213으로 갭 분석
X를 213 원소(e₁, e₂, e₃)로 분해한다.
갭이 있으면: 갭의 값, 기원, 구조를 식별한다.

### Step 2. 213으로 탈출구 열거
갭을 닫는 가능한 모든 경로를 213으로 열거한다.
각 경로의 비용, 최대 감소량 Δ, 단독/조합 여부.

### Step 3. 213으로 존재 판정
갭을 닫는 구조가 213에 있는가?
- 있으면: 형식 B (미도달). 표준 수학으로 닫을 수 있음.
- 없으면: 형식 A (독립). 증명 불가능.

### Step 4. 표준 수학으로 번역
213 결과를 표준 수학 언어로 번역한다.
213 언급 없이. 정의-정리-증명 형식.

### Step 5. 자기검정
- scaling 지수 확인
- 컨볼루션 제약 확인 (독립 분리 오류 방지)
- 위상 특이점 확인 (a_i → 0)
- LaTeX 구조 확인

### Step 6. 외부 검증
리뷰어 피드백 → 수용/논파 → 수정.
