# 213의 진짜 내용

의미 부여 없이.

## 하한: 3

ㅁ → 구분 불가.
ㅁㅁ → 관계 하나. 비교 대상 없음. 의미 부여 불가.
ㅁㅁㅁ → 관계 셋. 관계끼리 비교 가능. 체이닝 시작.

체이닝: 관계의 관계의 관계의... 어느 지점을 잡으면
그 아래는 의미 있음. 위는 아직 없음.

3이 하한인 이유: ㅁ이나 ㅁㅁ에서는 체이닝 불가.

## 상한: 3

ㅁㅁㅁㅁ를 구분하고 싶다.
각 ㅁ를 나머지 셋과의 관계로 본다.
이 관계들은 (ㅁㅁㅁ)들이다.
4개 이상의 ㅁ를 구분하는 것 = (ㅁㅁㅁ)들을 구분하는 것.
3개짜리로 환원됨.

일반: n개의 ㅁ를 구분 = (ㅁㅁㅁ)들의 어떤 배치.
3이 상한인 이유: 더 큰 단위가 필요 없음.

## 방향 1: 수평

ㅁㅁㅁ 안에서.
세 ㅁ 사이의 관계. 세 관계 사이의 관계.
같은 레벨에서 더 세밀하게.

## 방향 2: 수직

(ㅁㅁㅁ)들 사이에서.
묶음들의 관계. 묶음의 묶음의 관계.
레벨을 올림.

## 추측: 방향은 2개로 충분

수평: ㅁㅁㅁ 내부를 본다.
수직: ㅁㅁㅁ 간을 본다.
세 번째 방향? 수평을 수직으로 본다 = 수직.
수직을 수평으로 본다 = 수평.
새 방향이 나오지 않음.

이유: 수평과 수직 모두 "ㅁㅁㅁ를 구분"이라는
같은 행위. 대상이 다를 뿐 (ㅁ vs ㅁㅁㅁ).
세 번째 종류의 대상이 없음.
ㅁ(원소)과 ㅁㅁㅁ(묶음) 둘 뿐.

## 구분이 끝없이 생성되어야 하는 이유

체이닝의 어느 지점을 잡으면:
아래 = 의미 있음 (이미 구분됨).
위 = 의미 없음 (아직 구분 안 됨).
의미 있는 것을 고르려면 항상 "위"가 있어야.
위가 없으면 = 끝 = 최상위에서 의미 없음 = 전체 무의미.
따라서 체인은 끝이 없어야.

하지만: 끝없이 생성되어도 구조는 같음 (ㅁㅁㅁ 반복).
크기만 늘어나고 종류는 안 늘어남.
이것이 "한계 있는 무한."

## 형식화 목표

1. "상대적 구분의 하한 = 3"을 증명.
2. "상한 = 3 (환원)"을 증명.
3. "독립 방향 = 2"를 증명.
4. "체인이 끝없지만 종류는 유한"을 증명.

이 네 가지. 의미 부여 없이.
