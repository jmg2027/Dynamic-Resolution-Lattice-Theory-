# 213

이 글은 선형 매체에 쓰여 있으므로 213을 왼쪽에서 오른쪽으로 나열했다.
이것은 매체의 제약이지 213의 성질이 아니다.

213은 자기 자신을 기술하는 데 필요한 최소 장비가 213이기 때문에 213이다.

## 편의 표기

이하 모든 표기는 213을 선형 매체에서 다루기 위한 편의이다.

**def +** := a1b를 a+b로 쓴다. 1이 사이에 있는 것.

**def ×** := ab를 a×b로 쓴다. 1이 사이에 없는 것.

**def 0** := 11. 경계 옆에 경계. 내용 없음.

**def =** := 231. 구분(2) 후 재귀 비교(3) 후 경계만 남음(1).

## 213의 구조

- 2: 구분 연산 (행위)
- 1: 경계/존재/구분 표지 (흔적)
- 3: 구분의 자기적용 (창발)

행위(2)가 흔적(1)을 남기고, 흔적들의 관계에서 새것(3)이 창발한다.

## 순서에 대하여

213의 위상적 구조는 순환이다. 선형 순서 2→1→3은 매체의 제약.
순환 구조에서 213 = 132 = 321 = 213. 시작점이 없다.

순서를 부여하면: 자유도가 추가되고 정보가 손실된다.

## 소거에 대하여

11이 1인가, 11인가, 무(ε)인가?
이 체계 안에서는 미결정이다. 순서와 마찬가지로, 소거 규칙은 선택이다.

- 소거 안 함 → 1, 11, 111, ... 전부 다름 → ℕ
- 11 → 1 → 모든 경계 동일 → 부울 대수
- 11 → ε → 쌍소거 → ℤ/2ℤ

각 선택이 다른 수학을 만든다. 어느 것도 213에서 강제되지 않는다.

## 자기기술성

- 2+3 = 213 (def +에 의해 문자 그대로)
- 213의 길이 = 3 (세 번째 자리의 값)
- = 은 231 (213의 순환 순열 = 같은 구조)
- 체계가 자기 자신의 동치 판정기

## C(3,2) = 3

n개에서 순서 없는 쌍의 수 C(n,2) = n(n-1)/2.
C(n,2) = n의 유일한 해: n = 3.

3은 쌍짓기의 유일한 고정점이다.
자기복제: 3개의 쌍 → 3개 → 3개의 쌍 → ...
