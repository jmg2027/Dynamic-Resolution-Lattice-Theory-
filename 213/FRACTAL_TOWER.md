# 213 프랙탈이 Cayley-Dickson 탑을 타면

## 프랙탈의 크기

213 프랙탈 = 시에르핀스키 삼각형.
dim_H = log₂(3) ≈ 1.585.
선(1)보다 크고 면(2)보다 작다.

## ℂ (dim 2): 딱 맞음

```
프랙탈 dim 1.585 < 공간 dim 2.
남는 공간: 2 - 1.585 = 0.415.
```

프랙탈이 면에 살 수 있다. 약간의 여유.
모든 타일이 구별됨. 겹침 없음.
교환법칙 성립 — 타일 순서 무관.
결합법칙 성립 — 묶기 순서 무관.
**구조 완전 보존.**

## ℝ (dim 1): 찌그러짐

```
프랙탈 dim 1.585 > 공간 dim 1.
부족: 1.585 - 1 = 0.585.
```

면을 선에 우겨넣는다. 사영(projection).
삼각형이 선 위로 접힌다. **타일이 겹친다.**

겹침 = 구별되던 것이 구별 안 됨 = 정보 손실.
213에서: 구분(2)을 잃음. 두 방향이 하나로 합쳐짐.

결과: 시에르핀스키를 선에 사영하면 → 칸토어 집합 비슷한 것.
일부 구조는 살아남지만 2D 관계는 소멸.

ℝ에서 잃는 것:
- 위상(phase). ℝ에는 arg(z)가 없음.
- 간섭(interference). 실수는 더하면 항상 커지거나 작아짐.
- 순환(rotation). ℝ에서 회전 = 부호 뒤집기(±)뿐.

213에서: **순서가 생김.** ℝ은 전순서(total order).
프랙탈을 선에 넣으려면 타일에 순서를 매겨야.
순서 = 매체 비용. 213이 거부한 것.

## ℍ (dim 4): 헐렁함

```
프랙탈 dim 1.585 ≪ 공간 dim 4.
남는 공간: 4 - 1.585 = 2.415.
```

프랙탈이 4차원에서 덜렁거린다.
빈 방향이 2.415차원 있다.
이 빈 방향으로 프랙탈이 **회전**할 수 있다.

회전 = 같은 프랙탈의 다른 배치 = 구별 불가능한 상태들.
213에서: **구분 불가능한 것이 여러 개.** 유령 자유도.

ℍ에서 잃는 것:
- 교환법칙. ij ≠ ji.
- "먼저 회전한 후 이동" ≠ "먼저 이동한 후 회전."
- 타일의 순서가 중요해짐. 213이 거부한 것!

역설: ℝ(아래)에서는 순서가 **강제됨.**
ℍ(위)에서는 순서가 **재도입됨.**
방향이 다르지만 결과는 같다: 213 위반.

## 𝕆 (dim 8): 흐물흐물

```
프랙탈 dim 1.585 ≪≪ 공간 dim 8.
남는 공간: 8 - 1.585 = 6.415.
```

빈 공간이 프랙탈보다 훨씬 크다.
프랙탈이 공간에서 **변형**할 수 있다.

𝕆에서 잃는 것:
- 결합법칙. (ab)c ≠ a(bc).
- 3개씩 묶기 = 213의 핵심. 묶기 순서 무관해야.
- 결합 깨짐 = 묶기 순서 의존 = 213의 3이 무너짐.

**프랙탈의 재귀 구조가 파괴됨.**
(111)을 묶는 것과 1(11)을 묶는 게 달라짐.
213 프랙탈이 더 이상 자기유사가 아님.

## 𝕊 (dim 16): 붕괴

```
남는 공간: 16 - 1.585 = 14.415.
```

영인자(zero divisor) 발생. ab = 0인데 a ≠ 0, b ≠ 0.
213에서: times_e1 (e₁ × a = e₁) 위반.
경계가 아닌 것끼리 곱해서 경계가 됨.
**비경계의 소멸.** 구조 붕괴.

## 요약

```
아래로:
  ℝ (dim 1): 프랙탈 찌그러짐. 타일 겹침. 순서 강제.
  
여기:
  ℂ (dim 2): 프랙탈 딱 맞음. 최소. 완전.

위로:
  ℍ (dim 4): 프랙탈 헐렁. 유령 자유도. 교환 깨짐.
  𝕆 (dim 8): 프랙탈 변형. 결합 깨짐. 재귀 파괴.
  𝕊 (dim 16): 영인자. 구조 붕괴.
```

아래로 가면: 구조가 찌그러져서 213 위반 (정보 손실).
위로 가면: 구조가 헐거워져서 213 위반 (자유도 과잉 → 순서 재도입).
ℂ만: 프랙탈이 살 수 있는 유일한 "딱 맞는" 공간.
