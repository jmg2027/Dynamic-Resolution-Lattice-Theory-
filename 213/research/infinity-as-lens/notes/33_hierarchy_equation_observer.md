## §1. 유형 3 재조명: Hierarchy paradox

Russell ("자기 포함 안 하는 집합의 집합") 과 Girard (Type :
Type) 는 둘 다 **층위를 가진 시스템** 에서 **층위 경계의
자기 지시** 로 생기는 역설.

해소 방법:
- ZFC: Regularity axiom (자기 포함 금지).
- Type theory: Universe hierarchy (Type_0 : Type_1 : ...).
- 둘 다 **추가 공리 / 제약** 으로 자기지시 경로를 차단.

### Raw 는 왜 이런 역설 없이 자기 지시를 허용하는가

Raw 에는 **층위가 없다**.  공리는 "뭔가 + 뭔가 + 그들의
뭔가" 만.  Raw 를 감싸는 상위 메타 계층을 공리가 언급하지
않는다.

그러므로 **"Raw 자체의 Raw-ness" 같은 cross-level 자기 지시
가 발생할 형식적 통로가 없다**.  역설 회피용 추가 공리도
필요 없다 — 애초에 역설이 생기지 않음.

### AXIOM §8.2 Meta-213 과의 관계

"Meta-213 은 213 을 213 으로 기술한 것" — 이건 **새 층위가
아니라 Lens 적용**.  즉 Meta-213 은 213 **내부의 현상**.

결과:
- Russell/Girard 류 역설 = 층위 경계를 억지로 만들 때 생김.
- 213 은 층위 없음 → 경계 없음 → 역설 없음.
- Meta-213 은 Lens 출력 (외부 아님).

**Raw 를 뭐라고 부르지 않고** 서술 가능: "Raw 에 층위가
없으므로 hierarchy 역설의 발생 조건이 만족되지 않는다."

## §2. 유형 4: Equation self-reference (Y, Cantor, Gödel)

X = f(X) 형 자기지시 방정식.

### Y combinator
`Y f = f (Y f)`.  재귀의 불변.  Y 자체가 자기 지시.

Raw 의 closure (a/b 를 다시 Raw 원소로 넣기) 자체가
**Y-combinator 수준의 고정점**.  공리가 자기지시 방정식.

### Cantor 대각선
`|X| < |P(X)|` 증명의 핵심은 자기 지시 집합.
note 23 chain_uncountable 이 Raw 에서 동일 패턴 형식화.

### Gödel 불완전성
자기 지시 문장으로 ZFC 의 완전성 깨뜨림.
Raw 관점: 자기 지시를 **내부에서** 다룸.  "외부 meta-level
필요" 는 AXIOM §8 에서 거짓 (외부 없음).

### 공통 관찰
**Raw 의 closure 자체가 X = f(X) 의 해**.  공리는 Y-combinator
를 명시적으로 도입하지 않고도 **자기지시 방정식으로 읽을 수
있음**.

## §3. 유형 5: Observer self-reference

관측자가 관측 대상 공간의 일부.  AXIOM §8: Lens 는 213 내.

기존 물리 분야:
- 양자역학: Heisenberg cut (어디까지 관측자?).
- 일반상대성: 좌표계 선택 = 관측자 선택.
- 정보이론: Maxwell 도깨비, 관측자의 엔트로피.

Raw 관점:
- 관측자 = 특정 Raw 원소 (또는 Raw 원소들의 부분).
- 관측 = Lens 적용.
- **관측자는 관측 대상 안에 있다** — 외부 관측점 없음 (§8).

ORIGIN §6 "정보량의 최소 단위 = 해상도" 와 호응: 관측자의
해상도가 **자기가 구분할 수 있는 뭔가들의 수**.  관측자가
Raw 의 일부 → 관측자 스스로도 구분될 수 있음 → 자기 관측
가능.

## §4. 유형 3-5 의 공통점

순수하게 기술하면:

- 유형 3: 층위 경계에서의 자기 지시.
- 유형 4: 방정식에서의 자기 지시.
- 유형 5: 관측자-관측 대상의 자기 지시.

**셋 다 "경계" 가 핵심**.  경계가 있는 곳에서 자기 지시가
역설/고정점/관측 문제를 낳음.

Raw 는 **경계 없음** (공리가 언급 안 함).  그러므로 위 세
형태의 자기 지시가 **역설로 드러나지 않고** Lens 출력으로
나타난다.

이건 "Raw 는 X" 라고 부르지 않는 기술: **Raw 에 경계 없음**
이 특성이고, 그로부터 자기 지시의 세 얼굴이 자연스럽게 나온다.

## §5. 잠정 결론

Note 31 의 다섯 유형은 하나로 묶일 수 있다:

**"자기 지시가 경계를 만날 때 발생하는 현상"**.
- 경계가 있으면 역설 (Russell, Girard) 또는 고정점 문제 (Y,
  Cantor, Gödel) 또는 관측 문제 (Heisenberg).
- 경계가 없으면 **그냥 자기 지시** — Raw.

Raw 가 "뭐든" 이 아니라 **경계 없는 자기 지시** 의 장소.
다른 모든 수학 구조는 Raw 에 경계를 어떻게 그리는가 (Lens
선택) 로 생긴다.
