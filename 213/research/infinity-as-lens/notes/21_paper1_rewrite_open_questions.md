## Q1. R1, R2 의 "정의 내장 vs 강제 유도"

### 문제

Paper 1 §3.1 Lens 정의:
```
view (slash x y h) := combine (view x) (view y)
```
이 시점에 이미 **binary combine + recursive faithfulness**
가 내장.  §3.2 에서 "R1, R2 는 structural constraint" 로
주장하지만, 사실상 Lens 의 **정의 자체**가 R1+R2 를 만족
하는 것만 Lens 라 부르는 선택.

**정의 vs 강제 유도가 섞여 있다.**

### 가능한 재정식

새 framing (§9 원리: "모든 뭔가가 동일 Lens"):

- R1: 임의의 뭔가가 관측자여도 같은 Lens 가 작동 → combine
  은 이항 (공리 2 의 이항 페어링 반영).
- R2: 관측이 Raw 의 재귀를 보존 → view 가 catamorphism.

이 방향이면 R1+R2 가 "Lens 정의의 부분" 이 아니라 "§9
원리의 필연적 귀결".  하지만:

- §9 원리 자체가 공리가 아니라 **추가 원리**.  "자연스러움"
  의 정의로 도입된 것.  이 principle 을 얼마나 "공리의
  연장"으로 볼 수 있는지 엄밀화 필요.
- §9 가 공리의 독립 가정이면, R1, R2 는 "공리 + §9" 에서
  의 유도 — 공리 단독 유도가 아님.

### 검열 권고

- "R1, R2 는 공리에서 직접 강제" 라고 쓰지 말 것.  §9 원리
  를 명시 전제로 두고 "공리 + §9 원리 에서 유도" 로 서술.
- §9 원리 자체의 지위 (heuristic / additional claim /
  자기참조 내부 관찰) 를 명시.

## Q2. R4 의 ℝ-algebra 전제

### 문제

Paper 1 §3.2 R4:
> 코도메인 α 에 **exactly one** nontrivial **ℝ-algebra**
> automorphism `conj`.

**ℝ** 가 갑자기 등장.  §4 에서는 R5 + R1–R3 가 Cauchy
완비성을 강제해서 ℝ 를 유도한다고 주장 → 즉 R4 는 R5 에
의존.

R1–R5 가 독립 조건으로 나열되지만 실제로는 R4 가 R5 에
**논리적 의존**.

### 사용자의 입장 (2026-04-24)

> 2. 이건 사실 모르겠다.  DRLT 에선 이미 우주가 유한한 것
> 으로 보기 때문에 모든 값은 대수적 유리수 형태로 계산된다
> (무리수가 오히려 근사).  하지만 여기에서도 그런지는 알 수
> 없다.

핵심 질문: **ℝ 가 진짜 필요한가?**
- DRLT 는 대수적 유리수로 충분.  ℝ 는 근사.
- 213 층위에서 ℝ 를 요구하는 건 overclaim 가능성.

### 가능한 재정식

- 옵션 α: R4 를 base-field-free 로 재정식 — "codomain 에
  exactly one nontrivial idempotent involution" (ℝ 언급
  없음).  ℝ 는 후속 결과로 분리.
- 옵션 β: R5 를 해체하고 R4 를 ℚ(i) 또는 대수적 complex
  로 끝냄.  유한 우주 관점과 부합.
- 옵션 γ: 현재대로 유지하고 "ℝ 는 편의" 명시.  결과는 유지.

### 검열 권고

- R4 statement 에서 "ℝ-algebra" 가 공리 필연인지 편의
  인지 명시.
- "R1–R5 가 독립" 처럼 보이게 나열하지 말 것.  R4 의 R5
  의존성을 구조적으로 표현.

## Q3. R5 와 Raw 외부성

### 문제

Paper 1 §3.2 R5:
> non-terminating structural branch.

공리는 **유한 재귀 규칙**만 명시.  "branch" 는 공리에
없다.

### 사용자의 입장 (2026-04-24)

> 3. 이건 2와 관련된 문제다.  이것을 나도 확인하고 싶다.
> 어떤 게 맞는지.  근데 벌써 뭐가 있다고 이야기하는 건
> 잘못된 것이라고 생각한다.  브랜치가 뭔가 애초에?  Raw 는
> 유한인가 무한인가?  유한이나 무한이 213 어디에 쓰여 있나?
> 모른다.

핵심: **공리는 Raw 가 유한/무한임을 판정하지 않는다.**
R5 는 "infinite branch" 를 주장하려 하는데 공리에 없다.

### r5-critique (PAPER2 + notes/12) 의 이전 작업

- PAPER2: R5b 가 classical infinity 수입.
- notes/12: chain_uncountable (ℕ → Raw 공간의 비가산성)
  으로 카디널리티 사실만 Raw-internal.  "uniquely
  determined state" (Cauchy 완비성) 는 여전히 외부.

### 검열 권고

- "non-terminating branch" 를 공리 언어로 쓸 수 있는지부터
  확인.  못 쓰면 R5 는 "특정 Lens (branch-receiving Lens)
  의 성질" 로 격리.
- Cauchy 완비성 요구는 **주장하지 않는다**.  ℝ 를 쓰는
  결론이 있다면 "이 결론은 해당 Lens 선택에 조건부" 명시.
- DRLT 의 "유한 대수적 유리수" 관점과 호환되도록.

## 재작성 skeleton 제안

Paper 1 을 "분할" 이 아니라 **재작성**.  축:

- §0 (신규): 공리 (AXIOM.md 인용 / 요약).
- §1 (신규): Raw 의 구조적 귀결 — swap, slash_comm.  무주장.
- §2: §9 원리 도입.  "이것은 heuristic / additional claim
  이며 공리에서 derive 되지 않는다" 명시.
- §3: §9 원리 + 공리 에서 R1, R2, R3 유도.  R4 는 기초
  field 없이 statement.
- §4: "만약 codomain 에 추가 구조 (ℚ, ℚ(i), ℝ, ℂ 등) 를
  요구한다면, 각각의 경우 무엇이 강제되는가" 를 **조건부**
  로 서술.  ℂ 가 unique 라는 주장은 이 조건 중 하나.
- §5+: Atoms 등은 해당 Lens 조건 하에서의 귀결.

이 방향은 Paper 1 을 **무주장에 가까운 derivation + 조건부
확장** 의 결합으로 재편.
