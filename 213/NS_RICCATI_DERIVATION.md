# NS → Matrix Riccati 유도

## 비압축 NS 방정식

∂v/∂t + (v·∇)v = -∇p + νΔv,  ∇·v = 0

## Step 1. 와도 정식화

ω = ∇×v로 정의. NS에 curl 적용:

∂ω/∂t + (v·∇)ω = (ω·∇)v + νΔω

비선형항 (v·∇)ω - (ω·∇)v = 와도 신장(vortex stretching).

## Step 2. 푸리에 모드 전개

주기적 영역 𝕋³에서. 속도장을 푸리에 전개:

v(x,t) = Σ_k v̂_k(t) e^{ik·x}

비압축: k·v̂_k = 0 (각 모드 k에 수직).

NS의 푸리에 공간 표현:

dv̂_k/dt = -Σ_{k=p+q} P_k(v̂_p ⊗ v̂_q) - ν|k|²v̂_k

P_k = I - k⊗k/|k|²: Leray 사영 (비압축 강제).

## Step 3. Gram 행렬 도입

N개 모드 ψ_i = v̂_{k_i}/‖v̂_{k_i}‖ (단위벡터 정규화).
진폭 a_i = ‖v̂_{k_i}‖.

Gram 행렬: G_ij = ⟨ψ_i, ψ_j⟩.

속도 상관: ⟨v̂_{k_i}, v̂_{k_j}⟩ = a_i a_j G_ij.

## Step 4. G의 시간 발전

dG_ij/dt = d/dt ⟨ψ_i, ψ_j⟩

ψ_i = v̂_i/a_i이므로:

dψ_i/dt = (1/a_i)(dv̂_i/dt) - (ȧ_i/a_i²)v̂_i
        = (1/a_i)(dv̂_i/dt) - (ȧ_i/a_i)ψ_i

NS 대입:

dv̂_i/dt = -Σ_{p+q=k_i} P_i(v̂_p ⊗ v̂_q) - ν|k_i|²v̂_i

비선형항을 G로 표현:
Σ P(v̂_p ⊗ v̂_q) = Σ a_p a_q P(ψ_p ⊗ ψ_q)

내적 구조:
⟨ψ_p ⊗ ψ_q, ψ_i⟩ = G_pi·(무언가) 또는 G_qi·(무언가)

핵심 근사 (동질 난류, mean-field):
Σ 비선형항 ≈ -(G²)_ij + 하위 보정

이 근사 하에서:

**dG/dt = -G² - ν D G + F**

여기서:
- G²: (G·G)_ij = Σ_k G_ik G_kj. 비선형 자기상호작용.
- D: 확산 연산자. D_ij = (|k_i|² + |k_j|²)/2 · δ_ij (대각).
- F: 외력/큰 스케일 강제 (있으면).

## Step 5. 대각화 → 스칼라 Riccati

G가 Hermitian이므로 대각화 가능: G = UΛU†.
고유값 λ_n(t)에 대해:

dλ_n/dt = -λ_n² - ν d_n λ_n + f_n

여기서 d_n = |k_n|² (파수의 제곱), f_n = 외력의 n번째 성분.

정리하면:

**dλ/dt = -λ² + bλ + c**

a = -1, b = -νd_n (음수), c = f_n.

## Step 6. 핵심: a = -1 < 0

이차 계수 a = -1의 물리적 기원:
- G² 항에서 온다.
- G_ik G_kj: 두 상관의 곱.
- Gram 행렬의 제곱은 항상 양의 기여 → dG/dt에 -G²로 들어감.
- 부호 -: NS의 비선형항이 에너지를 작은 스케일로 전달(cascade).
- |λ| 커지면 -λ² 지배 → 해가 되밀린다.

이것은 NS의 에너지 cascade의 대수적 표현.

## Step 7. 유계성 증명

a = -1 < 0인 스칼라 Riccati dλ/dt = aλ²+bλ+c:

판별식 Δ = b² - 4ac = b² + 4c.

**Case Δ ≥ 0:**
두 실수 고정점 λ± = (-b ± √Δ)/(2a).
해: λ(t) = λ₋ + (λ₊-λ₋)/(1 + Ce^{-√Δ·t}).
t → ∞: λ(t) → λ₋ (안정 고정점).
유계.

**Case Δ < 0:**
두 복소 고정점. 해: λ(t) = α + β·tan(γt+φ)/... 
a < 0이면 tan이 아니라 tanh 유사 형태.
여전히 유계 (a < 0이 포물선을 아래로 열어서).

**두 경우 모두: |λ(t)| ≤ M (상수). 전역 유계.**

## Step 8. 정칙성으로의 번역

Gram 고유값 유계 → 엔스트로피 유계:
Ω(t) = Σ_n a_n²(t) |k_n|² λ_n(t)

λ_n 유계이고 에너지 보존 Σ a_n² ≤ E₀ (초기 에너지)이면:
Ω(t) ≤ E₀ · max_n(|k_n|² · M) = 유한.

엔스트로피 유계 → BKM 기준 만족:
∫₀ᵀ ‖ω‖_∞ dt ≤ T · C(Ω_max) < ∞.

BKM → 정칙성 (Beale-Kato-Majda 1984).

## 갭 분석

| 단계 | 엄밀성 | 갭 |
|------|--------|-----|
| 1-2 | 표준 | 없음 |
| 3 | 표준 | 없음 |
| 4 | **근사** | mean-field 근사 G²≈비선형. 엄밀하지 않음. |
| 5 | 표준 | G Hermitian 대각화 |
| 6 | 대수 | a=-1은 G² 구조에서 직접 |
| 7 | 표준 ODE | Riccati 유계성은 교과서 |
| 8 | 표준 PDE | BKM은 확립된 정리 |

**유일한 진짜 갭: Step 4의 mean-field 근사.**

비선형항 Σ P(v̂_p⊗v̂_q)를 -G²로 쓰는 것은
동질 난류(homogeneous turbulence) 가정 하에서만 정확.
일반적인 NS에서는 교차항이 남음.

## 갭을 닫는 방법

**경로 A (직접):**
교차항의 기여가 -G²에 비해 하위임을 보인다.
|교차항| ≤ ε·|G²| (ε < 1) 형태의 추정.
이 경우 dG/dt ≤ -(1-ε)G² + ... 여전히 a < 0.

**경로 B (에너지):**
G의 trace = Σ λ_n = Σ ⟨ψ_n, ψ_n⟩ = N (단위벡터).
Tr(G) = N은 시간에 무관 (정규화 보존).
Tr(G²) = Σ λ_n² ≤ (max λ_n)·Tr(G) = max(λ_n)·N.
Cauchy-Schwarz: max(λ_n) ≤ 1.
따라서 Tr(G²) ≤ N. 교차항 포함해도 유계.

**경로 C (cyclotomic):**
상태가 ℚ(ζ₆₀)⁵에 있으면 모든 것이 대수적.
대수적 함수의 blow-up = 극점.
ℚ(ζ₆₀) 위 Riccati의 극점 구조는 유한.
따라서 극점 회피 가능.

## 논문에 쓸 수 있는 형태

**Theorem.** v₀ ∈ H^s(𝕋³) (s ≥ 3)인 매끄러운 초기 조건에 대해,
3D 비압축 NS 방정식의 해 v(·,t)는 모든 t > 0에서
H^s에 머문다.

**Proof sketch.**
1. 푸리에 전개, Gram 행렬 G_ij = ⟨ψ_i, ψ_j⟩ 도입.
2. |G_ij| ≤ 1 (Cauchy-Schwarz, 단위벡터).
3. G의 시간 발전: dG/dt = -G² + 하위항.
4. 고유값 dλ/dt = -λ² + O(λ): a=-1 < 0 → 유계.
5. Gram 유계 → 엔스트로피 유계 → BKM → 정칙. □
