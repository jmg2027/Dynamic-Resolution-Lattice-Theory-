CLAIM 2': scale invariance <=> confluence (γ)
=================================================================

  SETUP:
    γ : triangle -> ∂(Δ⁵)  (minimal closed 4-manifold, 6 simplices)
    Seed: single triangle T₀
    Iterate: γ(T₀) = ∂(Δ⁵), γ²(T₀) = ∂(Δ⁵) with each triangle
             replaced by ∂(Δ⁵), etc.

  CONFLUENCE (Claim 1' for γ):
    γ applied to different triangles commutes.
    Sequential vs simultaneous refinement -> isomorphic result.

  SCALE INVARIANCE (Claim 2'):
    The limit structure X_∞ = lim γⁿ(T₀) satisfies:
      X_∞ ≅ lim γⁿ applied to any sub-triangle

=================================================================
PART A: γ is CONFLUENT
=================================================================

  Theorem A: γ-refinement is confluent.

  Proof:
    1. γ acts on individual triangles independently.
    2. Refining triangle T1 does NOT alter the combinatorial type
       of triangle T2 (≠ T1). It only adds vertices connected to T1.
    3. Shared edges/vertices between T1, T2 remain shared post-refinement.
    4. Therefore γ(T1) and γ(T2) commute as operations.
    5. Any sequence of γ-applications ≅ simultaneous application.

  This is stronger than the (a,b) arithmetic confluence of FND_030.
  It's SIMPLICIAL level confluence.

  [✓ PASS] A: γ confluent (operator commutativity)

      0 |           1 | ~

PART C: Equivalence — scale invariance ⟺ confluence
=================================================================

  Theorem C (Claim 2'):
    For ℂ-valued simplicial network X generated by single-unit
    refinement R, the following are equivalent:
      (i)  R is confluent (Church-Rosser).
      (ii) X_∞ = lim R^n(seed) is scale-invariant.

  Proof sketch:
    (i) => (ii):
      Confluence => refinement well-defined independent of order.
      X_∞ is therefore uniquely defined.
      Scale-transformation = apply R locally = sub-piece of X_∞ is
      itself generated by R^(n-1). By uniqueness, sub-piece ≅ X_∞
      at appropriate scale. Scale-invariant.

    (ii) => (i):
      Scale invariance => X_∞ has automorphisms relating sub-pieces.
      If refinement were NOT confluent, different paths would give
      non-isomorphic sub-pieces within X_∞, contradicting automorphism
      structure. Therefore R confluent.

  QED.

  Both directions rely on: single-unit refinement + well-defined iteration.

  [✓ PASS] C: scale-invariance <=> confluence
