## The narrative

### 1. The atomic foundation (Steps 1-5)

213's universe has three atomic constants:

- `NS = 3` — number of "spatial" generators
- `NT = 2` — number of "temporal" generators
- `d = 5 = NS + NT` — atomic dimension

These are not parameters; per C2 atomic constants (closed in
`theory/physics/foundations/atomic_constants.md`), they are the
unique solution to the 213-self-consistency equations.

`UniverseChain/Atomicity.lean` packages the atomic numerals;
`Universe.lean` ties them to `N_U = d^(d²) = 5²⁵` (per
`seed/RESOLUTION_LIMIT_SPEC.md`).

### 2. Nat213 as the typed atomicity (Step 6)

`Lens/Number/Nat213/Core.lean` constructs `Nat213` — a 213-native
natural-number type with **exactly NS = 3 constructors** for the
"spatial" part and **NT = 2 constructors** for the "temporal" part:

```
Nat213 ≅ Free magma generated by (NS spatial ctors, NT temporal ctors)
       ≅ Raw quotiented by chart-relativity (per seed/AXIOM/09)
```

The constructor count *is* the atomic 4-tuple — Nat213 is the
**typed witness** of (NS, NT, d).

### 3. Möbius matrix P (Step 7)

The Möbius generator of 213 is:

```
P = [[2, 1], [1, 1]]
```

Encoding atomicity:
- `top-left = 2 = NT`
- `trace = 3 = NS`
- `det = 1 = glue = NS − NT`

The "glue" `1 = NS − NT` is the **Lorentz-style boost generator**
in 213's physics reading.

`Lib/Math/Mobius213.lean` + `Mobius213OneAsGlue.lean` formalize
this; `theory/Raw/Mobius.lean` was promoted to this location per
the M-series cleanup.

### 4. Lucas seeds (Step 8)

The Lucas sequence with seeds `L_0 = NT = 2, L_1 = NS = 3` gives:

```
L_2 = L_0 + L_1 = 5 = d (atomic dimension)
L_3 = 7 = Mersenne M_3 = −χ(K_{3,2}^{(c=2)})  (the gluon stress)
L_4 = 12, L_5 = 19, ...
```

The Lucas-Mersenne dual at 7 is **the same 7** that appears as
the Euler characteristic stress of K_{3,2}^{(c=2)} — the
algebra-side and topology-side fingerprints of the same atomicity
.

### 5. Pentagonal closure (Step 9)

Working mod 5:

```
P^5 ≡ −I (mod 5)
P^10 ≡ I (mod 5)
```

The Möbius iterator **closes in 10 steps** mod 5 (period 10 =
2·5 = NT·d).  Geometrically: the spiral generated by `P` starting
at `(NS, NT) = (3, 2)` forms a **pentagonal pattern** with
dihedral D_5 symmetry.

`RotationGeometry.lean` (in `Lens/Number/Nat213/`) closes this
explicitly: 25 theorems on Pell-Fib, pentagonal closure, Lucas /
Mersenne.

### 6. SL(2, F_5) ≅ binary icosahedral 2I (Step 10)

The mod-5 Möbius group `SL(2, F_5)` is **isomorphic to the binary
icosahedral group 2I** of order 120 = 24·5.  This is the
algebraic realization of the pentagonal closure.

K_{3,2}^{(c=2)} cohomology connects via `H¹ = ℤ^8` — the same 8
that is the SU(3) gluon octet (C3 chain, `theory/physics/symmetry/
c3_chain.md`).

`Lens/Number/Nat213/AlgebraicGeometry.lean` closes the algebraic-
geometric core (17 theorems): SL(2, F_5) ≅ 2I, K_{3,2}^{(2)}
cohomology, dual fillings, CRT structure.

### 7. Dual fillings (Step 11)

The Euler characteristics of Δ⁴ and K_{3,2}^{(c=2)} are dual:

```
χ(Δ⁴) + χ(K_{3,2}^{(c=2)}) = 1 + (−7) = −6
                            = −(NS·NT)
                            = −(Type C ZOmega units, |ZOmega^×| = 6)
```

The two cohomological substrates are **complementary fillings** of
the same atomicity — Δ⁴ contributes `+1` (contractibility),
K_{3,2}^{(c=2)} contributes `−7` (8 holes − 1 component).

`UniverseChain/BipartiteFractal.lean` + `Decomposition.lean`
formalize the bipartite-fractal structure.

### 8. CRT decomposition (Step 12)

The full universal-chain closure works via Chinese Remainder Theorem:

```
mod 5 (pentagonal D_5):    P^10 ≡ I, order 10
mod 2 (triangular S_3):    order 3
lcm = 30 = full closure period
```

The two prime moduli (5 = d and 2 = NT) generate the full
universe-chain dynamics; CRT splits the action cleanly.

`UniverseChain/MobiusChain.lean` is the sentinel importing all
chain steps + the CRT decomposition.
