# Non-holonomicity as escaping the finite-state machine

**Triggering question.**  A real's continued fraction `(a·µ¢)` is *non-holonomic* when no linear
recurrence with polynomial coefficients governs it ‚Äî the conjectured status of œÄ.  What does
that *mean* structurally, why is it the residue's own signature read one scale up, and where is
its practical edge?

## The claim

> **Holonomic = generated by a finite-state machine (bounded memory).  Non-holonomic = a trace
> that escapes every finite-state machine.**  œÄ's (conjectured) continued-fraction
> non-holonomicity is the residue's *source-without-enclosure* ‚Äî `CoResidue.spineL_escapes` ‚Äî read
> at the continued-fraction scale.

A finite linear recurrence emits the next term from a finite window: bounded memory, a
finite-state transition.  Two machine classes name the two natural finite transitions, and the
residue escapes their union.

## Derivation by citation

**The foundational shape.**  Read as an automaton, the self-pointing residue is the
free-running counter trace `spineL`: a genuine behaviour (`CoResidue.spineL_antiRefl`) that is
the trace of **no finite reachable state** (`CoResidue.spineL_escapes`,
`FlatOntologyClosure.object1_not_surjective`).  The finite carrier `Raw` reads as a *reachable,
reduced, deterministic* machine (`StateMachine.mu_carrier_reachable_reduced_machine`); exactly
one behaviour ‚Äî the counter ‚Äî escapes it (`StateMachine.counter_no_return`,
`residue_escapes_minimal_machine`).  "Escapes every finite-state machine" is the residue's
defining move.

**The two machine classes, at the sequence scale.**  Over `‚Ñ§` ‚Äî the readout group in which the
difference `Œî s n = s(n+1) ‚àí s n` closes (`integers_as_difference_lens.md`) ‚Äî the bounded-memory
sequences split:

  - **time-varying, linear** ‚Äî `ZeroRunNonHolonomic.HomogRec`: a homogeneous recurrence
    `lead(n)¬∑a(n+k) = R(n, window)` with `lead` eventually nonzero (every integer P-recursive
    sequence satisfies one).  Escape certificate: `zero_run_not_homogRec` ‚Äî a sequence with
    arbitrarily long zero-runs and infinitely many nonzero terms satisfies none (a zero window
    forces, via `lead ‚âÝ 0`, every later term to zero).
  - **time-invariant, arbitrary** ‚Äî `ZeroRunNonHolonomic.AutoRec`: a fixed window-rule `F`,
    `n`-independent ‚Äî the classical deterministic finite-state machine.  Two escape certificates,
    one per density: *sparse* ‚Äî `two_continuations_not_autoRec` (general form
    `distinct_next_equal_window_not_autoRec`): a window pattern with two distinct continuations
    is emitted by no such machine (the Myhill‚ÄìNerode kernel: one state cannot yield two outputs);
    *dense* ‚Äî `MorseHedlund.aperiodic_not_autoRec`: **any** non-eventually-periodic `Bool`-sequence
    (even one with no long runs and no exhibited repeated window) escapes the class ‚Äî a
    `k`-step-deterministic bounded sequence has finitely many windows and a deterministic shift,
    so pigeonhole forces eventual periodicity (`kStepDet_periodic`, the Morse‚ÄìHedlund bounded
    direction).  The characterization is exact and closes to an **iff**
    (`MorseHedlund.bool_autoRec_iff_evPeriodic`): the autonomous machine class over `Bool` is
    *precisely* the eventually-periodic sequences ‚Äî the converse `bool_evPeriodic_autoRec` builds
    the machine outright (period `p`, threshold `N` ‚üπ window length `N+p` and the single-slot rule
    `F w = w N`, since `a(n+N+p)=a(n+N)`).  So escaping `AutoRec` is *equivalent* to aperiodicity,
    no slack.

  The two classes unite as `ZeroRunNonHolonomic.FiniteRecurrence := HomogRec ‚à® AutoRec` ‚Äî the
  finite-state-machine class; a sequence is non-holonomic in the strong, machine-free sense when
  it satisfies neither.

**The witnesses.**  The growth direction: `(n!)‚Åø` outruns every bounded-memory envelope
(`NonHolonomicWitness.superFact_nonHolonomic`, the elementary Klazar bound).  The bounded,
sparse direction: the indicator of the powers of two, `œá`, escapes **both** machine classes ‚Äî
`ZeroRunNonHolonomicWitness.{chi_nonHolonomic, chi_not_autoRec}`, bundled as
`chi_not_finiteRecurrence : ¬¨ FiniteRecurrence œá` (an all-zero length-`k` window is continued by
`0` mid-run and by `1` just before a power of two).  The bounded, *dense* direction: **every**
bounded aperiodic sequence escapes `AutoRec` (`aperiodic_not_autoRec`), witnessed concretely by
`isPow2_morse_not_autoRec` and applying in principle to Thue‚ÄìMorse / Sturmian.  `œá` is bounded
yet needs unbounded memory: the powers-of-two trace escapes every finite-state machine, the
CF-scale shadow of `spineL_escapes`.  And the dense axis is non-vacuous: **Thue‚ÄìMorse** ‚Äî
run-length `‚â§ 2` (`ThueMorseAperiodic.tm_run_le_two`: no three consecutive equal values, so the
*sparse* `zero_run_not_homogRec` route, which needs arbitrarily long constant runs, provably
cannot catch it ‚Äî only the dense route can), no exhibited repeated window ‚Äî is proven aperiodic
‚àÖ-axiom by
the self-similar period-descent (`ThueMorseAperiodic.tm_not_evPeriodic`: an even period halves
through `tm(2n)=tm(n)`, an odd period drops by one through `tm(2n+1)=¬¨tm(n)`, and period `1`
contradicts the always-differing consecutive pair `tm(2n),tm(2n+1)`), so
`ThueMorseAperiodic.tm_morse_not_autoRec` is a concrete dense inhabitant of
`aperiodic_not_autoRec`.

The *location* is sharp and worth stating exactly.  `œá` escapes **both** machine classes
(`chi_not_finiteRecurrence`) because its long zero-runs feed the elementary `HomogRec`
certificate.  Thue‚ÄìMorse escapes the autonomous class `AutoRec` ‚Äî but whether it escapes the
time-varying `HomogRec` (P-recursive) class is **not** ‚àÖ-axiom-closable by the present tools, and
not by accident: `tm_run_le_two` *forbids* the long constant runs that `zero_run_not_homogRec`
needs, so the only route to `¬¨ HomogRec` for a no-long-run sequence is the deep
*automatic ‚àß aperiodic ‚üπ non-holonomic* theorem (Cobham/Christol), which has no elementary
constructive shadow.  So the two canonical witnesses sit on *different* sides of the same map: `œá`
is caught by the cheap zero-run mechanism on both axes; Thue‚ÄìMorse is caught only by the dense
automaton route, and its P-recursive escape shares œÄ's open status.  Density is not a strengthening
of the sparse witness ‚Äî it is the regime where the cheap certificate is structurally unavailable.

**Why the tier is a ring stratification.**  The finite-Œî-depth `‚Ñ§`-sequences form a ring
(`FiniteDepthAlgebra.polyDepthZ_mul`: depths add under product) with exact normal form
(`NewtonGregory.reconstruct`: a depth-`d` sequence *is* `Œ£_{i‚â§d}(Œî‚Å±s 0)¬∑binom(¬∑,i)`) and
polynomial growth bound (`NewtonGregory.poly_bound`).  The holonomicity tier measures where a
partial-quotient sequence sits relative to this generating ring; the genuine non-holonomic tier
is outside it ‚Äî no finite generating machine.

The automatic side furnishes a concrete escapee.  Read off the same automaton that emits
Thue‚ÄìMorse, the digit counter `s2 = popcount` is **unbounded** (`ThueMorseAperiodic.s2_unbounded`:
`s2(ones k) = k`) yet returns to its minimum `1` at every power of two (`s2_pw2 : s2(2^k)=1`), so
it is *not eventually monotone* ‚Äî proven ‚àÖ-axiom (`s2_not_eventually_monotone`).  By the standard
fact that a finite-Œî-depth integer sequence is eventually a polynomial in `n` ‚Äî hence eventually
monotone ‚Äî `popcount` carries **no** finite difference-depth: the automatic counter sits *outside*
the generating ring while its bounded `{0,1}` readout `tm` is the dense non-holonomic witness.  The
two facts are one structure seen at its two ends: the bounded readout escapes the *machine*
(term-window), the unbounded counter escapes the *ring* (finite Œî-depth).  The 213-native half ‚Äî
non-monotonicity ‚Äî is closed ‚àÖ-axiom; the bridge `polyDepth d s ‚üπ eventually monotone` is the one
classical step still to formalise ‚Äî the faithful-`Int`, both-sign analogue of
`PositiveFloorUnbounded.positive_floor_unbounded`, whose vanishing-top-difference case is exactly
where `popcount`'s sub-linear growth hides the oscillation.

## Two finite-state notions, diverging at Thue‚ÄìMorse

"Finite-state" is ambiguous and Thue‚ÄìMorse is exactly where the two readings split.

  - **Finite memory in the previous *terms*** ‚Äî the term-window machine `AutoRec`: the next value
    is a fixed function of a length-`k` window of past outputs.  Thue‚ÄìMorse **lacks** this
    (`tm_morse_not_autoRec`).
  - **Finite memory in the *digits of the index*** ‚Äî a base-`k` automaton (DFAO) reading the
    digits of `n` and emitting a state-output.  Thue‚ÄìMorse **has** this:
    `ThueMorseAperiodic.tm_eq_popParity` proves `tm n = ` parity of the binary digit-sum
    `popcount(n)` ‚Äî a two-state automaton on the base-`2` digits (the digit-sum recurrence
    `s2(2n)=s2(n)`, `s2(2n+1)=s2(n)+1` carries one bit of parity, `succ_parity`).

The classical theory living on this split: a `k`-automatic sequence is **P-recursive (holonomic)
iff it is eventually periodic** (Cobham-type), and over `GF(p)` is exactly the algebraic-power-
series class (Christol).  So *automatic ‚àß aperiodic ‚üπ non-holonomic*: Thue‚ÄìMorse, being
digit-automatic (`tm_eq_popParity`) and aperiodic (`tm_not_evPeriodic`), is non-holonomic at the
generating-function scale ‚Äî the term-window escape `tm_morse_not_autoRec` is the elementary,
‚àÖ-axiom shadow of that boundary, with the *automatic* hypothesis now also formalised.

**Dyadic self-similarity.**  The automaticity has a visible fold: on `[2^k, 2^{k+1})` Thue‚ÄìMorse is
the *bitwise complement* of `[0, 2^k)` ‚Äî `tm(2^k + n) = ¬¨tm(n)` for `n < 2^k`
(`tm_shift_pow`), because the disjoint high bit `2^k` adds exactly one to the digit-sum and flips
its parity (`s2_add_pow : s2(2^k+n) = s2 n + 1`).  This is the self-similar doubling that *is* the
2-automaton: the next dyadic block is the current one negated, so the whole sequence is generated
by folding, never by a finite window of past terms.

**One automaton, two output maps.**  The sparse and dense witnesses are not two automata but one,
read differently.  Both `œá`/`isPow2` (powers of two) and `tm` (Thue‚ÄìMorse) are output functions of
the *same* `2`-automatic digit-sum `s2 = popcount`: `isPow2 n = (s2 n == 1)`
(`isPow2_eq_s2_one` ‚Äî a power of two has exactly one binary `1`) and `tm n = (s2 n % 2 == 1)`
(`tm_eq_popParity`).  Their holonomic statuses differ ‚Äî `œá` escapes *both* machine classes (its
`s2 = 1` output map carves the sparse powers-of-two set with long zero-runs), `tm` only the
autonomous one (its parity output map gives run-length `‚â§ 2`) ‚Äî but the divergence is entirely in
the *output map* applied to the shared automaton, not in the automaticity.  This is the concrete
texture of "automatic ‚àß aperiodic": the automaton is fixed; what the readout does to its state is
what decides which machine class the sequence escapes.

## Dual function

Classically this is the **holonomic / P-recursive** dividing line and its automata-theoretic
twin (a C-finite sequence over `GF(2)` *is* a linear-feedback shift register; a deterministic
finite automaton cannot emit two outputs from one state ‚Äî Myhill‚ÄìNerode).  The 213 reading
sharpens *what* is escaping (the residue) and *which structure* it escapes (the difference-Lens
generating ring), and grounds the slogan **"holonomicity is a property of the pointing, not the
number"** (`PresentationDependence.crossDetSmall_is_presentation_dependent`, `rcut_rescale`): a
pointing *is* a choice of machine for the same residue; the residue is the behaviour all of them
converge to and none generates as a finite state.

## Cross-frame

One fact in four frames: the residue's `spineL_escapes` (foundational) ¬∑ `chi_not_autoRec`
(automata / FSM) ¬∑ `chi_nonHolonomic` (P-recursive / D-finite) ¬∑ `poly_bound` outside the
finite-depth ring (difference calculus).  The ¬µF/ŒΩF pair ‚Äî reachable finite states vs the
escaping trace ‚Äî is not a dichotomy but one self-pointing act read at its two cofinal ends.

## The practical edge

The line *has a finite generating machine* / *does not* is where computation lives.

  - **Computing œÄ = choosing a holonomic pointing.**  Record computations use the Chudnovsky
    hypergeometric series ‚Äî a holonomic pointing admitting binary splitting ‚Äî because the
    continued fraction, being non-holonomic, is useless for fast computation.  Same number, two
    pointings; one computes through the holonomic one.  This is the working principle of
    creative telescoping / the holonomic systems approach: holonomic objects have finite
    descriptions a computer-algebra system can store, sum, and certify.  (Classical, cited.)
  - **Sequence generation and a usable test.**  C-finite over `GF(2)` = an LFSR (CRC, stream
    ciphers, PRNGs); the non-holonomic witnesses admit no such finite recurrence.
    `zero_run_not_homogRec` is a ready criterion ‚Äî *arbitrarily long zero-runs + infinitely many
    nonzero ‚üπ no homogeneous linear recurrence* ‚Äî so a "guess-the-recurrence" search can be
    abandoned with proof that none exists.
  - **Interpolation.**  `NewtonGregory` is the exact forward-difference calculus ‚Äî Newton
    forward-difference interpolation and Newton‚ÄìCotes integration; a data sequence is degree-`d`
    polynomial-interpolatable iff its difference table bottoms out at depth `d`.

## Constructive accessibility

The essay lands on syntactic objects: `chi_not_autoRec : ¬¨ AutoRec œá` and
`chi_nonHolonomic : ¬¨ HomogRec œá` ‚Äî `œá = ` the indicator of the powers of two, a total
computable `0/1` function (`chi_values`), proven to escape both finite-state machine classes ‚Äî
and, on the dense axis, an **actual continued fraction**: `tmCF`, the `{1,2}`-valued Thue‚ÄìMorse
partial-quotient sequence (`tmCF_ge_one : 1 ‚â§ a‚Çô`, `tmCF_mem : a‚Çô ‚àà {1,2}`), the CF of a concrete
real whose partial quotients are bounded, dense, aperiodic, and generated by no autonomous machine
(`tmCF_not_autoRec`, via the shift-invariance `autoRec_shift`).  The dense non-holonomic axis is no
longer abstract ‚Äî it carries a continued fraction one can write down.

The ‚àÖ-axiom reach is now complete across both machine classes and both densities:

  | machine class | escape certificate | witnesses |
  |---|---|---|
  | time-varying homogeneous P-recursive (`HomogRec`) | `zero_run_not_homogRec` | `(n!)‚Åø`, `œá` |
  | time-invariant autonomous, sparse (`AutoRec`) | `two_continuations_not_autoRec` | `œá` |
  | time-invariant autonomous, dense (`AutoRec`) | `aperiodic_not_autoRec` (Morse‚ÄìHedlund) | any bounded aperiodic ‚Äî `isPow2`, and the genuinely-dense **Thue‚ÄìMorse** (`ThueMorseAperiodic.tm_morse_not_autoRec`, run-length `‚â§ 2`) |

## Open frontier

  - œÄ's *own* continued-fraction non-holonomicity is classically open and not closable
    ‚àÖ-axiom: the analytic shape obstruction (Flajolet‚ÄìGerhold‚ÄìSalvy) has no constructive shadow
    and bottoms out at the unproven Gauss‚ÄìKuzmin normality of œÄ.  The escape is *certified* here
    only for constructed witnesses, not for œÄ.
  - The mod-2 forbidden-factor obstruction (Garrabrant‚ÄìPak) is, for its concrete witnesses
    (Champernowne, which contains `0^k` for every `k`), already subsumed by the zero-run and
    Myhill‚ÄìNerode certificates ‚Äî no 2-adic valuation needed; the full structural statement (every
    P-recursive sequence omits a mod-2 factor, including the no-zero-run corner) is strictly more
    general but witnessed by nothing the present tools miss.
  - The dense axis is now non-vacuous: **Thue‚ÄìMorse** (run-length `‚â§ 2`) is formalised and
    proven aperiodic ‚àÖ-axiom (`ThueMorseAperiodic`), so `aperiodic_not_autoRec` has a concrete
    genuinely-dense inhabitant ‚Äî not merely the long-run `isPow2`.  Sturmian (complexity `n+1`)
    remains the next dense instance, awaiting its own aperiodicity.
