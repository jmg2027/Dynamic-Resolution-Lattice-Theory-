## 2. 213-native Emergence

### 2.1 Raw axiom = atomicity logic

- a ≠ b: primitive negation
- slash distinguishes: primitive disjunction (Xor)
- Bool has two values: only *the base* of classical logic (LEM not asserted)

`lean/E213/Lens/SemanticAtom.lean` already partial — Prop is an instance.

### 2.2 Cut = predicate

`Cut := Nat → Nat → Bool` is *predicate* form.  Bool function.
Sigma logic = 213-native predicate calculus.

### 2.3 Intuitionistic logic

213 is *constructive* — every theorem must show a *witness*.  Without
Choice, LEM (every proposition true or false) is also not automatic.  *Intuitionistic*
logic in 213-native form.

### 2.4 Gödel incompleteness limitations

213's 4 clauses are *finite* — avoid incompleteness?
Reality: Lean 4's own Type theory is powerful enough → Gödel applies.
But incompleteness *over* 213 is separate.

### 2.5 Proof = trajectory

Proof = bisection trajectory.  Each step = decision.  *Proof = path*
in dyadic tree.

All theorems in 213 propEq = *finite path*.
