## 2. 213-native 등장

### 2.1 Raw axiom = atomicity logic

- a ≠ b: 원시적 negation
- slash distinguishes: 원시적 disjunction (Xor)
- Bool 이 두 값: classical logic *의 base* 만 (LEM 명시 X)

`Research/SemanticAtom.lean` 이 이미 일부 — Prop도 instance.

### 2.2 Cut = predicate

`Cut := Nat → Nat → Bool` 이 *predicate* 형식.  Bool function.
Sigma logic = 213-native predicate calculus.

### 2.3 직관주의 logic

213 은 *constructive* — 모든 정리는 *witness* 보여야.  Choice
없으니 LEM (모든 명제 true 또는 false) 도 자동.  *intuitionistic*
logic 의 213-native form.

### 2.4 Gödel incompleteness 한계

213 의 4 clause 가 *유한* — incompleteness 회피?
실제: Lean 4 자체 Type theory 가 충분히 강력 → Gödel 적용.
하지만 213 *위* 에서의 incompleteness 는 별도.

### 2.5 Proof = trajectory

Proof = bisection trajectory.  각 step = 결정.  *증명 = path*
in dyadic tree.

213 의 모든 정리 propEq = *유한 path*.
