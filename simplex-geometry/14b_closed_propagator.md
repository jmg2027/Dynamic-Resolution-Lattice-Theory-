## Free vs Confined 전파자

### Free 입자 (심플렉스 밖으로 전파 가능)

```
P_free(k) = (1 + 2αk/d) / (1 + αk/d)

전개 파라미터: α_GUT = 6/(25π²)
Sector factor: f = k/d, k = 선택한 꼭지점 수

k=1: f=1/5  (전자, 뉴트리노 — 1-vertex 선택)
k=2: f=2/5  (전자 ℂ² — 2 시간 꼭지점)
k=3: f=3/5  (양성자 — 3 공간 꼭지점)
```

| 입자 | k | f | 오차 |
|------|---|---|------|
| m_p | 3 | 3/5 | **0.000%** |
| m_e | 2 | 2/5 | 0.185% |
| m_τ | — | (d-1)/d | 0.057% |
| m_c | — | 1/d | 0.020% |

### Confined 입자 (심플렉스 안에 갇힘)

```
P_conf = (1 − 2ε) / (1 − ε)

전개 파라미터: ε = α^{2/3}(1+α) ≈ 0.086
Sector factor: f = d/d = 1 (전체 심플렉스가 감옥)
```

| 입자 | 상태 | f | 오차 |
|------|------|---|------|
| m_u | confined | 1 | 1.06% (실험 ±12%) |

### 통합 관계

```
Confined = Free with {α → −ε, k → d}

Free:     P(k) = (1 + 2αk/d) / (1 + αk/d)
Confined: P    = (1 − 2ε)    / (1 − ε)
              = P(k=d) with α replaced by −ε
```

**감옥의 세 가지 효과:**
1. 결합 상수: α → ε = α^{2/3}(1+α) (더 강한 결합)
2. 전파 범위: k/d → d/d = 1 (전체 심플렉스가 경로)
3. 부호 반전: + → − (이웃으로 못 나감 → 경로 반전)

## δ_SSS = 180° = π: 올바른 값 (EXP_047b 확정)

### 이전 오류의 수정

13_universal_correction.md에서 δ_SSS = 90°로 주장했으나, 이는 **틀림**.

EXP_047b 변분 계산: δ_SSS = **180°** (모든 w, θ에서 일관).

이유: ∂(5-simplex)에서 SSS hinge를 둘러싸는 3개 simplex의 이면각:
- σ₃ = {A₁A₂A₃B₂B₃}: θ = 45° (|⟨B₂|B₃⟩|² = 1/2)
- σ₄ = {A₁A₂A₃B₁B₃}: θ = 45° (|⟨B₁|B₃⟩|² = 1/2)
- σ₅ = {A₁A₂A₃B₁B₂}: θ = 90° (⟨B₁|B₂⟩ = 0)
- 합: 180° → δ = 360° - 180° = **180°**

C²에서 3개 직교 벡터는 불가능 (dim=2). δ_SSS = 90°는 불가.

### (1+2x)/(1+x)는 δ=180°에서 자연스럽게 나온다

```
Regge holonomy: e^{iδ} (single loop around hinge)

δ = π → e^{iπ} = -1

자기에너지:
  Σ = x·1 + x²·(-1) + x³·(+1) + ...
    = x - x² + x³ - ... = x/(1+x)

전파자:
  P = 1 + Σ = (1+2x)/(1+x)  ← 양성자 0.000%!
```

이전 유도의 오류: "cos(2δ)=-1" 사용 (round-trip e^{2iδ} 가정).
올바른 유도: **single loop holonomy e^{iδ}** → δ=π에서 직접 -1.

이유: SU(5) 표현론에서의 AAA 참여도.
- u_R = AA pair (10 rep) → AAA 삼각형에 **두 꼭지점** 갇힘 → 완전 구속
- d_R = A vertex (5̄ rep) → AAA에 **한 꼭지점**만 참여 → 부분적 자유

**u quark이 d quark보다 가벼운 이유:**
u가 더 강하게 구속(ε-결합, f=1)되어 질량이 더 많이 깎임.
