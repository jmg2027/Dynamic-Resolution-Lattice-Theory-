## Free vs Confined 전파자

### Free 입자 (심플렉스 밖으로 전파 가능)

```
P_free(k) = (1 + 2αk/d) / (1 + αk/d)

전개 파라미터: α_GUT = 6/(25π²)
Sector factor: f = k/d, k = 선택한 꼭지점 수

k=1: f=1/5  (전자, 뉴트리노 — 1-vertex 선택)
k=2: f=2/5  (전자 ℂ² — 2 시간 꼭지점)
k=3: f=3/5  (양성자 — 3 공간 꼭지점)
```

| 입자 | k | f | 오차 |
|------|---|---|------|
| m_p | 3 | 3/5 | **0.000%** |
| m_e | 2 | 2/5 | 0.185% |
| m_τ | — | (d-1)/d | 0.057% |
| m_c | — | 1/d | 0.020% |

### Confined 입자 (심플렉스 안에 갇힘)

```
P_conf = (1 − 2ε) / (1 − ε)

전개 파라미터: ε = α^{2/3}(1+α) ≈ 0.086
Sector factor: f = d/d = 1 (전체 심플렉스가 감옥)
```

| 입자 | 상태 | f | 오차 |
|------|------|---|------|
| m_u | confined | 1 | 1.06% (실험 ±12%) |

### 통합 관계

```
Confined = Free with {α → −ε, k → d}

Free:     P(k) = (1 + 2αk/d) / (1 + αk/d)
Confined: P    = (1 − 2ε)    / (1 − ε)
              = P(k=d) with α replaced by −ε
```

**감옥의 세 가지 효과:**
1. 결합 상수: α → ε = α^{2/3}(1+α) (더 강한 결합)
2. 전파 범위: k/d → d/d = 1 (전체 심플렉스가 경로)
3. 부호 반전: + → − (이웃으로 못 나감 → 경로 반전)

## d quark은 free

d quark (5̄, A-type vertex):
- f = 1/3 (free propagator): 오차 0.21% ✓
- f = 1 (confined propagator): 오차 8.85% ✗

**d quark은 free, u quark은 confined.**

이유: SU(5) 표현론에서의 AAA 참여도.
- u_R = AA pair (10 rep) → AAA 삼각형에 **두 꼭지점** 갇힘 → 완전 구속
- d_R = A vertex (5̄ rep) → AAA에 **한 꼭지점**만 참여 → 부분적 자유

**u quark이 d quark보다 가벼운 이유:**
u가 더 강하게 구속(ε-결합, f=1)되어 질량이 더 많이 깎임.

## 물리적 의미

### QFT와의 대응

| QFT | DRLT |
|-----|------|
| Dyson equation | 등비급수의 닫힌 합 |
| 무한 Feynman diagram 재합산 | 분수 하나: (1+2x)/(1+x) |
| self-energy Σ(p²) | 전파자 루프의 deficit angle 위상 |
| renormalization | 자동 (급수가 수렴) |
| UV divergence | 없음 (이산 → 유한 항) |

### 왜 재규격화가 작동하는가

연속 QFT에서 재규격화가 필요한 이유:
무한대가 나타나서 수동으로 제거해야 함.

심플렉스 기하학에서: 급수가 **자연스럽게 수렴**.
각 항의 비율 = -x (with |x| < 1 always).
등비급수의 합 = (1+2x)/(1+x). **유한. 닫힌 형태.**

> Feynman이 20년 걸려 만든 재규격화의 이산 버전이 분수 하나.
