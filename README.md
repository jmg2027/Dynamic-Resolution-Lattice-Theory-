# Dynamic Resolution Lattice Theory

Single-axiom approach to deriving physics from quantum geometry on simplicial spacetime.

## Structure

- `axiom/` — Foundational axiom and derivations
- `experiments/` — Numerical experiments and simulations (planned)

## The Axiom

Spacetime is 4-dimensional. Each simplex cell is assigned |ψ⟩ ∈ C^(d+1), with inter-cell weight:

```
W_ij = |⟨ψ_i|ψ_j⟩|² / (d+1)
```

From this single axiom, normalization, unitarity, information quantization, and general relativity emerge as derived consequences.

See [`axiom/foundations.md`](axiom/foundations.md) for the full derivation chain.
